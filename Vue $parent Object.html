Vue $parent Object

Vue $parent பொருளைப் பற்றி அறிந்து கொள்ளுங்கள்

Vue $parent பொருள்

$parent பொருள் பெற்றோர் கூறின் Vue உதாரணத்தைக் குறிக்கிறது.

ரூட் கூறில் $parent பொருள் பயன்படுத்தப்பட்டால், $parent பொருளின் மதிப்பு null ஆக இருக்கும்.

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

குழந்தை கூறில் $parent பொருளைப் பயன்படுத்தி, பெற்றோர் கூறில் உள்ள 'text' தரவுப் பண்பை மாற்றுவது.

Example

<template>
  <div>
    <h3>Change Text</h3>
    <p>Click the button to toggle the text in the PRE tag of the parent component.</p>
    <button v-on:click="this.$parent.text='Hello!'">Change text in parent</button>
  </div>
</template>

💡 குறிப்பு:

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

ரூட் கூறில் $parent பொருள் பயன்படுத்தப்பட்டால், $parent பொருளின் மதிப்பு null ஆக இருக்கும்.

குழந்தை கூறிலிருந்து நேரடியாக பெற்றோர் உதாரணத்தை அணுக $parent பொருளைப் பயன்படுத்தலாம், முறைகளை அழைக்க, தரவுப் பண்புகளைப் படிக்க அல்லது மாற்றியமைக்க, மற்றும் பல.

⚠️ கவனிக்க:

Vue கூறுகளுக்கு இடையேயான தொடர்புக்கு props/emit அல்லது provide/inject ஐப் பயன்படுத்துவதைக் கவனியுங்கள், ஏனெனில் இவை வெளிப்படையாக வரையறுக்கப்பட்ட தொடர்பு வழிகளைக் கொண்ட குறியீடு பராமரிக்க எளிதானது.

மேலும் எடுத்துக்காட்டுகள்

Example

குழந்தை கூறில் $parent பொருளைப் பயன்படுத்தி, பெற்றோர் கூறில் உள்ள ஒரு முறையைக் குறிப்பிடுவது.

Template

<template>
  <div>
    <h3>Toggle Color</h3>
    <p>Click the button to toggle the color in the P tag of the parent component.</p>
    <button v-on:click="this.$parent.toggleColor">Toggle</button>
    <p>The 'toggleColor' method is also in the parent component.</p>
  </div>
</template>

பயிற்சி

❓ Vue $parent பொருள் எதைக் குறிக்கிறது?

குழந்தை கூறின் Vue உதாரணத்தை
✗ தவறு! $parent பொருள் குழந்தை கூறை அல்ல, பெற்றோர் கூறைக் குறிக்கிறது
பெற்றோர் கூறின் Vue உதாரணத்தை
✓ சரி! $parent பொருள் குழந்தை கூறிலிருந்து பெற்றோர் கூறின் Vue உதாரணத்தைக் குறிக்கிறது
ரூட் கூறின் DOM முனையை
✗ தவறு! DOM முனையைக் குறிப்பது $el பொருள் ஆகும், $parent பொருள் அல்ல
அடுத்தடுத்த கூறுகளின் பட்டியலை
✗ தவறு! $parent பொருள் அடுத்தடுத்த கூறுகளைக் குறிக்காது, பெற்றோர் கூறை மட்டுமே குறிக்கும்
