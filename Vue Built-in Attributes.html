Vue Built-in Attributes

Vue உள்ளமைக்கப்பட்ட பண்புகளைக் கற்றுக்கொள்ளுங்கள்

பண்புகள் அட்டவணை

Vue இன் முக்கிய உள்ளமைக்கப்பட்ட பண்புகள் பற்றிய விரிவான பட்டியல்:

பண்பு விளக்கம்
is தற்போதைய செயலில் உள்ள டைனமிக் கூறை குறிக்கிறது
key v-for உடன் உருவாக்கப்பட்ட கூறுகளுக்கு தனிப்பட்ட மதிப்பை வழங்குகிறது
ref டெம்ப்ளேட்டில் உள்ள ஒரு கூறுக்கு குறிப்பை வழங்குகிறது, இதனால் பின்னர் ஸ்கிரிப்ட்டில் கூறை கையாள முடியும்

💡 முக்கிய குறிப்பு:

இந்த பண்புகள் Vue கட்டமைப்பின் ஒருங்கிணைந்த பகுதியாகும். அவை Vue உடன் வேலை செய்யும் போது மிகவும் பயனுள்ளதாக இருக்கும்.

Vue பண்புகள் பயிற்சி

Vue உள்ளமைக்கப்பட்ட பண்புகள் பற்றிய உங்கள் புரிதலைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue இல் v-for லூப்புடன் 'key' பண்பு எவ்வாறு பயன்படுத்தப்படுகிறது?

<div v-for="item in items" key="item">
✗ தவறு! விசை மதிப்பு தனித்துவமாக இருக்க வேண்டும், பொதுவாக ஒரு ID
<div v-for="item in items" :ref="item.id">
✗ தவறு! இது ref பண்பு, key பண்பு அல்ல
<div v-for="item in items" :key="item.id">
✓ சரி! இது v-for லூப்பில் key பண்பைச் சரியாகப் பயன்படுத்துகிறது
<div v-for="item in items" is="item.id">
✗ தவறு! இது is பண்பு, key பண்பு அல்ல
