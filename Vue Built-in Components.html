Vue Built-in Components

Vue உள்ளமைக்கப்பட்ட கூறுகளைக் கற்றுக்கொள்ளுங்கள்

Vue உள்ளமைக்கப்பட்ட கூறுகள்

Vue.js கட்டமைப்புக்கு பல சக்திவாய்ந்த உள்ளமைக்கப்பட்ட கூறுகள் உள்ளன. இந்த கூறுகள் சிக்கலான UI பணிகளை எளிதாக்குவதற்கும், செயல்திறனை மேம்படுத்துவதற்கும், கூடுதல் செயல்பாடுகளை வழங்குவதற்கும் வடிவமைக்கப்பட்டுள்ளன.

🏗️ முக்கிய புள்ளி:

இந்த உள்ளமைக்கப்பட்ட கூறுகள் Vue இன் ஒருங்கிணைந்த பகுதியாகும். அவை எந்த கூடுதல் நிறுவலும் இல்லாமல் உடனடியாகப் பயன்படுத்தப்படலாம்.

கூறுகள் அட்டவணை

Vue இன் முக்கிய உள்ளமைக்கப்பட்ட கூறுகள் பற்றிய விரிவான பட்டியல்:

கூறு விளக்கம்
<KeepAlive> செயலில் இல்லாத டைனமிக் கூறுகளின் நிலையை நினைவில் வைக்கிறது
<Suspense> அசின்க்ரோனஸ் கூறுகளுடன் பயன்படுத்தப்படுகிறது, அசின்க்ரோனஸ் கூறு ரிசால்வ் செய்ய காத்திருக்கும் போது ஃபால்பேக் ஸ்லாட்டின் உள்ளடக்கத்தை ரெண்டர் செய்கிறது, இதனால் டிஃபால்ட் ஸ்லாட்டின் உள்ளடக்கத்தை ரெண்டர் செய்ய முடியும்.
<Teleport> ஒரு எலிமென்ட்டை DOM கட்டமைப்பில் மற்றொரு இடத்திற்கு நகர்த்துகிறது
<Transition> v-if அல்லது v-show உடன், அல்லது டைனமிக் கூறுகளுடன் நமது பயன்பாட்டிலிருந்து நீக்கப்படும் அல்லது சேர்க்கப்படும் போது ஒரு எலிமென்ட்டை அனிமேட் செய்கிறது.
<TransitionGroup> v-for உடன் நமது பக்கத்தில் சேர்க்கப்படும் எலிமெண்ட்ஸை அனிமேட் செய்கிறது

⚠️ எச்சரிக்கை:

<Suspense> உள்ளமைக்கப்பட்ட கூறு சோதனை முறையில் உள்ளது, மேலும் உலகளவில் ஆதரிக்கப்படவில்லை.

Vue உள்ளமைக்கப்பட்ட கூறுகள் பயிற்சி

Vue உள்ளமைக்கப்பட்ட கூறுகள் பற்றிய உங்கள் புரிதலைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue இல் ஒரு கூறை DOM இல் வேறு இடத்திற்கு நகர்த்த எந்த உள்ளமைக்கப்பட்ட கூறு பயன்படுத்தப்படுகிறது?

✗ தவறு! <Move> என்பது Vue இல் உள்ளமைக்கப்பட்ட கூறு அல்ல
<Transport>
✗ தவறு! Vue இல் <Transport> கூறு இல்லை

✓ சரி! <Teleport> கூறு DOM இல் கூறுகளை வேறு இடத்திற்கு நகர்த்த பயன்படுகிறது

✗ தவறு! <Relocate> என்பது Vue இல் உள்ளமைக்கப்பட்ட கூறு அல்ல

சிறந்த நடைமுறைகள்

Vue உள்ளமைக்கப்பட்ட கூறுகளைப் பயன்படுத்தும் போது இந்த வழிகாட்டுதல்களைப் பின்பற்றவும்:

டைனமிக் கூறுகளின் நிலையைப் பாதுகாக்க <KeepAlive> பயன்படுத்தவும்
மோடல்கள் மற்றும் டூல்டிப்களுக்கு <Teleport> பயன்படுத்தவும்
எளிய அனிமேஷன்களுக்கு <Transition> பயன்படுத்தவும்
வரிசை அனிமேஷன்களுக்கு <TransitionGroup> பயன்படுத்தவும்
ப்ராடக்ஷன் பயன்பாடுகளில் <Suspense> பயன்படுத்துவதைத் தவிர்க்கவும் (இது இன்னும் சோதனை முறையில் உள்ளது)
