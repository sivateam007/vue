Vue 'emits' Option

🏁 உதவிக்குறிப்பு:

உங்கள் Vue சேவையகத்தைப் பெறுங்கள்

கூறிலிருந்து வெளியிடப்படும் தனிப்பயன் நிகழ்வுகளை அறிவிக்க emits விருப்பத்தைப் பயன்படுத்துதல்.

Example

export default {
  emits: ['custom-event'],
  methods: {
    notifyParent() {
      this.$emit('custom-event','Hello! ')
    }
  }
}

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்

வரையறை மற்றும் பயன்பாடு

emits விருப்பம் ஒரு கூறு எந்த தனிப்பயன் நிகழ்வுகளை வெளியிடுகிறது என்பதை ஆவணப்படுத்த பயன்படுகிறது.

emits விருப்பம் தேவையில்லை, அதாவது ஒரு கூறு emits விருப்பத்திற்குள் வரையறுக்காமல் நிகழ்வுகளை வெளியிட முடியும்.

emits விருப்பம் தேவையில்லை என்றாலும், இன்னும் கொண்டிருப்பது பரிந்துரைக்கப்படுகிறது, இதனால் பிற நிரலாளர்கள் கூறு எதை வெளியிடுகிறது என்பதை எளிதாகப் பார்க்க முடியும்.

emits விருப்பம் ஒரு வரிசையாகக் கொடுக்கப்பட்டால், வரிசையானது சரங்களாக emits பெயர்களைக் கொண்டுள்ளது. (மேலே உள்ள எடுத்துக்காட்டைப் பார்க்கவும்.)

emits விருப்பம் ஒரு பொருளாகக் கொடுக்கப்பட்டால், ஒரு பண்பு பெயர் ஒரு emit பெயர், மற்றும் மதிப்பு ஒரு செல்லுபடியாக்கும் செயல்பாடு அதற்கு இருந்தால், அல்லது emit க்கு செல்லுபடியாக்கும் செயல்பாடு இல்லையென்றால் 'null'. (கீழே உள்ள எடுத்துக்காட்டைப் பார்க்கவும்.)

மேலும் எடுத்துக்காட்டுகள்

Example

விருப்பங்களுடன் பொருள்களாக props ஐப் பயன்படுத்துதல், இதனால் பெற்றோர் கூறு வழங்காதபோது இயல்புநிலை உணவு விளக்கம் காட்டப்படும்.

FoodItem.vue:

<template>
    <div>
        <h2>{{ foodName }}</h2>
        <p>{{ foodDesc }}</p>
    </div>
</template>

<script>
export default {
    props: {
        foodName: {
            type: String,
            required: true
        },
        foodDesc: {
            type: String,
            required: false,
            default: 'This is the food description...'
        }
    }
};
</script>

App.vue:

<template>
  <h1>Food</h1>
  <p>Food description is not provided for 'Pizza' and 'Rice', so the default description is used.</p>
  <div id="wrapper">
    <food-item 
      food-name="Apples" 
      food-desc="Apples are a type of fruit that grow on trees."/>
    <food-item 
      food-name="Pizza"/>
    <food-item 
      food-name="Rice"/>
  </div> 
</template>

❓ Vue கூறுகளில் emits விருப்பத்தின் முக்கிய நோக்கம் என்ன?

பெற்றோர் கூறிலிருந்து தரவைப் பெற
✗ தவறு! இது props விருப்பத்திற்கு
கூறின் உள் நிலையை நிர்வகிக்க
✗ தவறு! இது data() செயல்பாட்டிற்கு
கூறின் ஸ்டைல்களை வரையறுக்க
✗ தவறு! emits விருப்பம் ஸ்டைல்களுடன் தொடர்பில்லாதது
கூறு வெளியிடும் தனிப்பயன் நிகழ்வுகளை ஆவணப்படுத்த
✓ சரி! emits விருப்பம் கூறு எந்த நிகழ்வுகளை வெளியிடுகிறது என்பதை ஆவணப்படுத்துகிறது
