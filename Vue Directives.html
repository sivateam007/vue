Vue டைரக்டிவ்களைக் கற்றுக்கொள்ளுங்கள்

Vue டைரக்டிவ்கள்

Vue க்கு ஒரு தொகுப்பு டைரக்டிவ்கள் உள்ளன.

டைரக்டிவ் விளக்கம்
v-bind ஒரு பண்பை ஒரு தரவு ப்ராபர்ட்டியுடன் பைண்ட் செய்கிறது
v-cloak தயாராகும் வரை ஒரு தொகுக்கப்படாத டெம்ப்ளேட்டை மறைக்கிறது
v-for உறுப்புகளின் பட்டியலை ரெண்டர் செய்கிறது
v-html டெம்ப்ளேட்டில் HTML குறியீட்டை வெளியிடுகிறது
v-if ஒரு உறுப்பை நிபந்தனையுடன் ரெண்டர் செய்கிறது
v-else-if if-ஸ்டேட்மென்ட்டின் முதல் பகுதி false ஆக இருக்கும் போது ஒரு உறுப்பை நிபந்தனையுடன் ரெண்டர் செய்கிறது
v-else if-ஸ்டேட்மென்ட்டின் முதல் பகுதி false ஆக இருக்கும் போது ஒரு உறுப்பை ரெண்டர் செய்கிறது
v-memo ஒன்று அல்லது அதற்கு மேற்பட்ட குறிப்பிடப்பட்ட ப்ராபர்ட்டிகளில் மாற்றம் கண்டறியப்படும் வரை ஒரு உறுப்பின் ரெண்டரிங் வரை நிறுத்தி வைக்கிறது
v-model ஒரு உள்ளீட்டு உறுப்புக்கும் தொடர்புடைய தரவு ப்ராபர்ட்டிக்கும் இடையே இருவழி பைண்டிங் உருவாக்குகிறது
v-on ஒரு இவெண்டை ஒரு செயலுடன் இணைக்கிறது
v-once ஒரு உறுப்பை ஒரு முறை மட்டுமே ரெண்டர் செய்கிறது
v-pre ஒரு உறுப்பு மற்றும் அதன் உள்ளடக்கத்தின் தொகுப்பை தவிர்க்கிறது
v-show ஒரு உறுப்பின் தோற்றத்தை நிபந்தனையுடன் மாற்றுகிறது
v-slot உள்ளடக்கத்தை ஒரு பெயரிடப்பட்ட ஸ்லாட்டிற்கு வழிநடத்துகிறது
v-text ஒரு உறுப்பின் உரை உள்ளடக்கத்தைப் புதுப்பிக்கிறது

💡 டைரக்டிவ்கள் பற்றி:

டைரக்டிவ்கள் Vue இல் சிறப்பு HTML பண்புகள் ஆகும், அவை v- என்ற முன்னொட்டுடன் தொடங்குகின்றன. அவை Vue க்கு DOM உறுப்புகளில் சிறப்பு நடத்தையைப் பயன்படுத்தச் சொல்லும். ஒவ்வொரு டைரக்டிவுக்கும் ஒரு குறிப்பிட்ட நோக்கம் உள்ளது, மேலும் அவை Vue பயன்பாடுகளை உருவாக்குவதற்கான கட்டிடத் தொகுதிகளாகும்.

டைரக்டிவ் வகைகள்

பைண்டிங் டைரக்டிவ்கள்

  • v-bind: பண்புகளை தரவுடன் இணைக்கிறது
  • v-model: இருவழி தரவு பைண்டிங்
  • v-text: உரை உள்ளடக்கத்தைப் புதுப்பிக்கிறது
  • v-html: HTML உள்ளடக்கத்தைப் புதுப்பிக்கிறது

கண்டிஷனல் டைரக்டிவ்கள்

  • v-if / v-else-if / v-else: நிபந்தனை ரெண்டரிங்
  • v-show: நிபந்தனை தோற்றம் மாற்றம்
  • v-for: பட்டியல் ரெண்டரிங்

இவெண்ட் டைரக்டிவ்கள்

  • v-on: இவெண்ட் ஹேண்ட்லிங்
  • v-once: ஒரு முறை மட்டும் ரெண்டர்
  • v-memo: மெமோயைசேஷன்

சிறப்பு டைரக்டிவ்கள்

  • v-cloak: தொகுப்பு மறைப்பு
  • v-pre: தொகுப்பு தவிர்ப்பு
  • v-slot: ஸ்லாட் ஹேண்ட்லிங்

பொதுவான பயன்பாடுகள்

v-bind: CSS ஸ்டைல்கள், கிளாஸ்கள் மற்றும் பிற HTML பண்புகளை டைனமிக் ஆக அமைக்க
v-model: படிவ உள்ளீட்டு புலங்களுக்கு இருவழி தரவு பைண்டிங்
v-for: வரிசைகள் அல்லது பொருள்களிலிருந்து உறுப்புகளின் பட்டியலை ரெண்டர் செய்ய
v-if / v-show: நிபந்தனையின் அடிப்படையில் உறுப்புகளைக் காட்ட அல்லது மறைக்க
v-on: பயனர் இடைமுகத்தில் இவெண்டுகளைக் கையாள (கிளிக், உள்ளீடு, முதலியன)

⚠️ பாதுகாப்பு எச்சரிக்கை:

v-html டைரக்டிவை கவனமாகப் பயன்படுத்தவும். இது HTML உள்ளடக்கத்தை நேரடியாக ரெண்டர் செய்கிறது, இது Cross-Site Scripting (XSS) தாக்குதல்களுக்கு வழிவகுக்கும். நம்பகமான மூலத்திலிருந்து மட்டுமே HTML உள்ளடக்கத்தை ரெண்டர் செய்ய v-html ஐப் பயன்படுத்தவும்.

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களை சோதிக்கவும்

❓ பயிற்சி:

படிவ உள்ளீட்டு புலத்திற்கும் தரவு ப்ராபர்ட்டிக்கும் இடையே இருவழி தரவு பைண்டிங் உருவாக்க எந்த Vue டைரக்டிவ் பயன்படுத்தப்படுகிறது?

v-bind
✗ தவறு! v-bind ஒரு வழி பைண்டிங் மட்டுமே வழங்குகிறது (தரவிலிருந்து HTML க்கு)
v-on
✗ தவறு! v-on இவெண்ட் ஹேண்ட்லிங்கிற்கானது, தரவு பைண்டிங்கிற்கு அல்ல
v-model
✓ சரி! v-model படிவ உள்ளீட்டு புலங்களுக்கு இருவழி தரவு பைண்டிங் வழங்குகிறது
v-text
✗ தவறு! v-text உரை உள்ளடக்கத்தைப் புதுப்பிப்பதற்கானது, இருவழி பைண்டிங்கிற்கு அல்ல
