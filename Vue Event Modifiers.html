Vue நிகழ்வு திருத்திகள்

Vue நிகழ்வு திருத்திகள்

Vue இல் உள்ள நிகழ்வு திருத்திகள் நிகழ்வுகள் முறைகளை இயக்கும் விதத்தை மாற்றியமைக்கின்றன மற்றும் நிகழ்வுகளை மிகவும் திறமையான மற்றும் நேரடியான வழியில் கையாள உதவுகின்றன.

நிகழ்வு திருத்திகள் Vue v-on வழிமுறையுடன் இணைந்து பயன்படுத்தப்படுகின்றன, எடுத்துக்காட்டாக:

  • HTML படிவங்களின் இயல்புநிலை submit நடத்தையைத் தடுக்க (v-on:submit.prevent)
  • ஒரு நிகழ்வு பக்கம் ஏற்றப்பட்ட பிறகு ஒரு முறை மட்டுமே இயக்க முடியும் என்பதை உறுதிப்படுத்த (v-on:click.once)
  • ஒரு முறையை இயக்க ஒரு நிகழ்வாக எந்த விசைப்பலகை விசையைப் பயன்படுத்த வேண்டும் என்பதைக் குறிப்பிட (v-on:keyup.enter)

v-on வழிமுறையை எவ்வாறு மாற்றுவது

ஒரு நிகழ்விற்கு எவ்வாறு எதிர்வினை செய்வது என்பதை மேலும் விரிவாக வரையறுக்க நிகழ்வு திருத்திகள் பயன்படுத்தப்படுகின்றன.

நாம் முன்பு பார்த்தது போல ஒரு குறிச்சொல்லை ஒரு நிகழ்வுடன் இணைப்பதன் மூலம் நிகழ்வு திருத்திகளைப் பயன்படுத்துகிறோம்:

<button v-on:click="createAlert">Create alert</button>

இப்போது, பொத்தான் கிளிக் நிகழ்வு பக்கம் ஏற்றப்பட்ட பிறகு ஒரு முறை மட்டுமே தீப்பிடிக்க வேண்டும் என்பதை மேலும் குறிப்பாக வரையறுக்க, .once திருத்தியை இவ்வாறு சேர்க்கலாம்:

<button v-on:click.once="createAlert">Create alert</button>

.once திருத்தியுடன் ஒரு எடுத்துக்காட்டு இங்கே:

எடுத்துக்காட்டு

'click' நிகழ்வு நடக்கும் முதல் முறையில் முறையை மட்டும் இயக்க .once திருத்தி <button> குறிச்சொல்லில் பயன்படுத்தப்படுகிறது.

<div id="app">
  <p>Click the button to create an alert:</p>
  <button v-on:click.once="createAlert">Create Alert</button>
</div>

    methods: {
      createAlert() {
        alert("Alert created from button click")
      }
    }
  })
  app.mount('#app')
</script>

வெவ்வேறு v-on திருத்திகள்

வெவ்வேறு சூழ்நிலைகளில் நிகழ்வு திருத்திகள் பயன்படுத்தப்படுகின்றன. விசைப்பலகை நிகழ்வுகள், சுட்டி கிளிக் நிகழ்வுகளைக் கேட்கும்போது நிகழ்வு திருத்திகளைப் பயன்படுத்தலாம், மேலும் ஒன்றுடன் ஒன்று சேர்த்து நிகழ்வு திருத்திகளைப் பயன்படுத்தலாம்.

நிகழ்வு திருத்தி .once விசைப்பலகை மற்றும் சுட்டி கிளிக் நிகழ்வுகள் இரண்டிற்கும் பிறகு பயன்படுத்தப்படலாம்.

விசைப்பலகை விசை நிகழ்வு திருத்திகள்

வெவ்வேறு விசைப்பலகை நிகழ்வு வகைகள் keydown, keypress, மற்றும் keyup ஆகியவை உள்ளன.

ஒவ்வொரு விசை நிகழ்வு வகையுடனும், ஒரு விசை நிகழ்வு நடந்த பிறகு சரியாக எந்த விசையைக் கேட்க வேண்டும் என்பதை நாம் குறிப்பிடலாம். எங்களிடம் .space, .enter, .w மற்றும் .up ஆகியவை உள்ளன.

ஒரு குறிப்பிட்ட விசையின் மதிப்பை நீங்களே கண்டறிய, வலைப்பக்கத்தில் அல்லது console.log(event.key) உடன் கன்சோலில் விசை நிகழ்வை எழுதலாம்:

keydown விசைப்பலகை நிகழ்வு 'getKey' முறையைத் தூண்டுகிறது, மேலும் நிகழ்வு பொருளிலிருந்து 'key' மதிப்பு கன்சோலில் மற்றும் வலைப்பக்கத்தில் எழுதப்படுகிறது.

<input v-on:keydown="getKey">
<p> {{ keyValue }} </p>
data() {
  return {
    keyValue = ''
  }
},
methods: {
  getKey(evt) {
    this.keyValue = evt.key
    console.log(evt.key)
  }
}

சுட்டி கிளிக் அல்லது விசை அழுத்தம் .alt, .ctrl, .shift அல்லது .meta போன்ற கணினி திருத்தி விசைகளுடன் இணைந்து நடக்கும்போது மட்டுமே நிகழ்வு நடக்கும் என்பதை நாம் தேர்வு செய்யலாம். கணினி திருத்தி விசை .meta விண்டோஸ் கணினிகளில் விண்டோஸ் விசையை அல்லது ஆப்பிள் கணினிகளில் கமாண்ட் விசையைக் குறிக்கிறது.

விசை திருத்தி விவரங்கள்
.[Vue key alias] மிகவும் பொதுவான விசைகளுக்கு Vue இல் அவற்றின் சொந்த மாற்றுப் பெயர்கள் உள்ளன:
  • .enter
  • .tab
  • .delete
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right
.[letter] நீங்கள் விசையை அழுத்தும்போது வரும் எழுத்தைக் குறிப்பிடவும். எடுத்துக்காட்டாக: 'S' விசையைக் கேட்க .s விசை திருத்தியைப் பயன்படுத்தவும்.
.[system modifier key] .alt, .ctrl, .shift அல்லது .meta. இந்த விசைகள் மற்ற விசைகளுடன் அல்லது சுட்டி கிளிக்குகளுடன் இணைந்து பயன்படுத்தப்படலாம்.

சுட்டி பொத்தான் திருத்திகள்

சுட்டி கிளிக்கு எதிர்வினை செய்ய, நாம் v-on:click என்று எழுதலாம், ஆனால் எந்த சுட்டி பொத்தான் கிளிக் செய்யப்பட்டது என்பதைக் குறிப்பிட .left, .center அல்லது .right திருத்திகளைப் பயன்படுத்தலாம்.

டிராக்பேட் பயனர்கள்: வலது கிளிக் உருவாக்க இரண்டு விரல்களுடன் கிளிக் செய்ய வேண்டியிருக்கலாம், அல்லது உங்கள் கணினியில் உள்ள டிராக்பேட்டின் கீழ் வலது பக்கத்தில் கிளிக் செய்ய வேண்டியிருக்கலாம்.

எடுத்துக்காட்டு

ஒரு பயனர் <div> உறுப்பில் வலது-கிளிக் செய்யும் போது பின்னணி நிறத்தை மாற்றவும்:

<div id="app">
  <div v-on:click.right="changeColor"
       v-bind:style="{backgroundColor:'hsl('+bgColor+',80%,80%)'}">
    <p>Press right mouse button here.</p>
  </div>
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        bgColor: 0
      }
    },
    methods: {
      changeColor() {
        this.bgColor+=50
      }
    }
  })
  app.mount('#app')
</script>

சுட்டி பொத்தான் நிகழ்வுகள் ஒரு கணினி திருத்தி விசையுடன் இணைந்தும் செயல்படலாம்.

முறையின் உள்ளே event.preventDefault() பயன்படுத்துவதன் மூலம் வலது கிளிக்குப் பிறகு துளி-கீழ் மெனு தோன்றுவதைத் தடுக்க சாத்தியமாகும், ஆனால் Vue .prevent திருத்தியுடன் குறியீடு மேலும் படிக்கக்கூடியதாகவும் பராமரிக்க எளிதானதாகவும் மாறும்.

படத்தை மாற்ற 'shift' விசைப்பலகை விசையைப் பிடித்து <img> குறிச்சொல்லில் இடது சுட்டி பொத்தானை அழுத்தவும்.

<div id="app">
  <p>Hold 'Shift' key and press left mouse button:</p>
  <img v-on:click.left.shift="changeImg" v-bind:src="imgUrl">
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        imgUrlIndex: 0,
        imgUrl: 'img_tiger_square.jpeg',
        imgages: [
          'img_tiger_square.jpeg',
          'img_moose_square.jpeg',
          'img_kangaroo_square.jpeg'
        ]
      }
    },
    methods: {
      changeImg() {
        this.imgUrlIndex++
        if(this.imgUrlIndex>=3){
          this.imgUrlIndex=0
        }
        this.imgUrl = this.images[this.imgUrlIndex]
      }
    }
  })
  app.mount('#app')
</script>

❓ பயிற்சி:

வலது கிளிக் செய்யப்படும் போது <div> உறுப்பு நிறம் மாறும் வகையில் சரியான குறியீட்டை வழங்கவும்.

மேலும், நீங்கள் ஒரு வலைப்பக்கத்தில் வலது கிளிக் செய்யும் போது தோன்றும் இயல்புநிலை துளி-கீழ் மெனு காட்டப்படாது என்பதற்கான குறியீட்டைச் சேர்க்கவும்.

<div id="app">
  <div v-on:click.="changeColor"
      v-bind:style="{backgroundColor:'hsl('+bgColor+',80%,80%)'}">
    <p>Press right mouse button here.</p>
  </div>
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        bgColor: 0
      }
    },
    methods: {
      changeColor() {
        this.bgColor+=50
      }
    }
  })
  app.mount('#app')
</script>
right
✗ தவறு! right மட்டும் போதாது, வலது கிளிக் மெனுவைத் தடுக்க prevent திருத்தியும் தேவை
left
✗ தவறு! left இடது கிளிக்கு, வலது கிளிக்கு அல்ல
right.prevent
✓ சரி! right வலது கிளிக்கு, prevent வலது கிளிக் மெனுவைத் தடுக்க
prevent.right
✗ தவறு! திருத்திகள் வரிசை முக்கியம், right முதலில், பின்னர் prevent
