Vue 'beforeUnmount' Lifecycle Hook

கூறு DOM இலிருந்து அகற்றப்படுவதற்கு சற்று முன் நிகழ்கிறது beforeUnmount வாழ்க்கைச் சுழற்சி கொக்கி

Example

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

<p> உறுப்புக்குள் இருந்து எடுக்கப்பட்ட உரையுடன் ஒரு எச்சரிக்கையை உருவாக்க beforeUnmount வாழ்க்கைச் சுழற்சி கொக்கியைப் பயன்படுத்துகிறது.

<script>
export default {
  beforeUnmount() {
    alert("beforeUnmount: The text inside the p-tag is: " + this.$refs.pEl.innerHTML);
  }
}
</script>

Definition and Usage

beforeUnmount வாழ்க்கைச் சுழற்சி கொக்கி கூறு DOM இலிருந்து அகற்றப்படுவதற்கு சற்று முன் நிகழ்கிறது.

beforeUnmount கொக்கியில் குறியீடு இயக்கப்படும் போது, கூறு இன்னும் முழுமையாக செயல்படுகிறது.

💡 குறிப்பு:

beforeUnmount கொக்கியில், கூறு இன்னும் DOM இல் உள்ளது, எனவே நீங்கள் இன்னும் DOM உறுப்புகளை அணுகலாம். இது கூறு அகற்றப்படுவதற்கு முன் சுத்தம் செய்தல் அல்லது முடிவு நிகழ்வுகளைச் செய்ய ஏற்ற இடமாகும்.
