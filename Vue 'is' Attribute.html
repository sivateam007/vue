Vue 'is' Attribute

Vue இல் 'is' பண்பைக் கற்றுக்கொள்ளுங்கள்

Vue 'is' பண்பு

Vue.js இல் 'is' பண்பு முக்கியமான உள்ளமைக்கப்பட்ட பண்புகளில் ஒன்றாகும். இது டைனமிக் கூறுகளைக் கையாளுவதற்கும், HTML கூறுகளை Vue கூறுகளுடன் மாற்றுவதற்கும் பயன்படுகிறது.

🎯 முக்கிய பயன்பாடு:

'is' பண்பு மூன்று முக்கிய விஷயங்களுக்குப் பயன்படுத்தப்படுகிறது: டைனமிக் கூறுகள், HTML கூறுகளை Vue கூறுகளுடன் மாற்றுதல் மற்றும் கஸ்டமைஸ்டு பில்ட்-இன் எலிமெண்ட்ஸ்.

எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

'is' பண்பு கம்ப்யூட்டட் மதிப்பு 'activeComp' உடன் v-bind (குறுக்குவழி :) மூலம் இணைக்கப்பட்டுள்ளது, இதனால் 'comp-one' கூறு அல்லது 'comp-two' கூறு காட்டப்படும்.

App.vue:

<template>
  <h1>Dynamic Components</h1>
  <p>App.vue switches between which component to show.</p>
  <button @click="toggleValue = !toggleValue">Switch component</button>
  <component :is="activeComp"></component>
</template>

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

'is' பண்பு மூன்று விஷயங்களுக்குப் பயன்படுத்தப்படலாம்:

டைனமிக் கூறுகள்: 'is' பண்பு உள்ளமைக்கப்பட்ட <component> எலிமென்ட்டில் டைனமிக் கூறை உருவாக்க அமைக்கப்பட்டுள்ளது, இங்கு 'is' பண்பு எந்த கூறு செயலில் இருக்க வேண்டும் என்பதை வரையறுக்கிறது.
ஒரு நேட்டிவ் எலிமென்ட்டை Vue கூறுடன் மாற்றுதல்: is="vue:my-component" ஒரு நேட்டிவ் HTML எலிமென்ட்டில் வைக்கப்பட்டு அதை Vue கூறுடன் மாற்றுகிறது.
கஸ்டமைஸ்டு பில்ட்-இன் எலிமென்ட்: கஸ்டமைஸ்டு பில்ட்-இன் எலிமெண்ட்ஸ் ஜாவாஸ்கிரிப்ட்டில் எழுதலாம், மேலும் 'is' பண்பு ஒரு HTML டேக்கில் அத்தகைய கஸ்டமைஸ்டு பில்ட்-இன் எலிமென்டாக வரையறுக்க பயன்படுத்தப்படலாம். இது Vue அம்சம் அல்ல.

⚠️ முக்கிய குறிப்பு:

நாம் vue: பிரிஃபிக்ஸைப் பயன்படுத்தவில்லை என்றால், அது கஸ்டமைஸ்டு பில்ட்-இன் எலிமென்டாக விளக்கப்படும், மேலும் Vue கூறு செருகப்படாது.

மேலும் எடுத்துக்காட்டுகள்

எடுத்துக்காட்டு 1

'is' பண்பைப் பயன்படுத்தி <img> டேக்கை Vue கூறுடன் மாற்றுதல்.

App.vue:

<template>
  <h2>Example Built-in 'is' Attribute</h2>
  <p>The IMG tag below is set to be replaced by a component by the use of 'is="vue:child-comp"'.</p>
  <img is="vue:child-comp" />
</template>

ChildComp.vue:

<template>
  <div>
    <h3>ChildComp.vue</h3>
    <p>This is the child component</p>
  </div>
</template>

<style scoped>
div {
  border: solid black 1px;
  background-color: lightgreen;
  padding: 10px;
  max-width: 250px;
  margin-top: 20px;
}
</style>

Vue 'is' பண்பு பயிற்சி

Vue இல் 'is' பண்பு பற்றிய உங்கள் புரிதலைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue இல் டைனமிக் கூறை உருவாக்க 'is' பண்பு எவ்வாறு பயன்படுத்தப்படுகிறது?

<dynamic :comp="activeComponent"></dynamic>
✗ தவறு! Vue இல் <dynamic> என்பது உள்ளமைக்கப்பட்ட டேக் அல்ல
<component :is="activeComponent"></component>
✓ சரி! இது Vue இல் டைனமிக் கூறுகளுக்கான சரியான தொடரியல்
<component is="activeComponent"></component>
✗ தவறு! இது v-bind (:is) இல்லாமல் உள்ளது, இது டைனமிக் மதிப்புகளுக்கு வேலை செய்யாது
<dynamic-component is="activeComponent"></dynamic-component>
✗ தவறு! Vue இல் உள்ளமைக்கப்பட்ட <component> டேக் பயன்படுத்தப்பட வேண்டும்

தொடர்புடைய பக்கங்கள்

கீழே உள்ள டுடோரியல்கள் மற்றும் குறிப்புகள் 'is' பண்பு பற்றிய மேலும் அறிவை வழங்கும்:

Vue டுடோரியல்: டைனமிக் கூறுகள்

Vue இல் டைனமிக் கூறுகளைப் பயன்படுத்துவது பற்றி மேலும் அறிக

Vue டுடோரியல்: Vue கூறுகள்

Vue கூறுகளை உருவாக்குதல் மற்றும் பயன்படுத்துதல் பற்றி அறிக

Vue டுடோரியல்: Vue கம்ப்யூட்டட் ப்ராபர்ட்டீஸ்

கம்ப்யூட்டட் ப்ராபர்ட்டீஸ் பற்றி அறிக

Vue டுடோரியல்: Vue v-bind டைரக்டிவ்

v-bind டைரக்டிவ் பற்றி அறிக

மேலும் குறிப்புகள்: Vue v-bind டைரக்டிவ் குறிப்பு, Vue <component> எலிமென்ட் குறிப்பு, Vue $refs ஒப்ஜெக்ட் குறிப்பு.
