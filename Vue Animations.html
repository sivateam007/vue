Vue அனிமேஷன்கள்

Vue இல் உள்ளமைக்கப்பட்ட <Transition> கூறு v-if, v-show அல்லது டைனமிக் கூறுகளுடன் உறுப்புகள் சேர்க்கப்படும் போது அல்லது நீக்கப்படும் போது அனிமேஷன்களைச் செய்ய உதவுகிறது.

மற்ற சந்தர்ப்பங்களில் பிளேன் CSS transitions மற்றும் animations ஐப் பயன்படுத்துவதில் தவறில்லை.

CSS Transition மற்றும் Animation குறுகிய அறிமுகம்

டுடோரியலின் இந்த பகுதி அடிப்படை CSS animations மற்றும் transitions பற்றிய அறிவைத் தேவைப்படுகிறது.

ஆனால் Vue குறிப்பிட்ட உள்ளமைக்கப்பட்ட <Transition> கூறைப் பயன்படுத்தி அனிமேஷன்களை உருவாக்குவதற்கு முன், Vue உடன் பிளேன் CSS animations மற்றும் transitions எவ்வாறு பயன்படுத்தப்படலாம் என்பதற்கான இரண்டு எடுத்துக்காட்டுகளைப் பார்ப்போம்.

🎯 எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

App.vue:

<template>
  <h1>Basic CSS Transition</h1>
  <button @click="this.doesRotate = true">Rotate</button>
  <div :class="{ rotate: doesRotate }"></div>
</template>

<script>
export default {
  data() {
    return {
      doesRotate: false
    }
  }
}
</script>

<style scoped>
  .rotate {
    rotate: 160deg;
    transition: rotate 1s;
  }
  div {
    border: solid black 2px;
    background-color: lightcoral;
    width: 60px;
    height: 60px;
  }
  h1, button, div {
    margin: 10px;
  }
</style>

CSS transition: 1 வினாடி

மேலே உள்ள எடுத்துக்காட்டில், <div> டேக் சுழலும் வகையில் ஒரு வகுப்பை வழங்க v-bind ஐப் பயன்படுத்துகிறோம். சுழற்சி 1 வினாடி எடுக்கும் காரணம், அது CSS transition பண்புடன் வரையறுக்கப்பட்டுள்ளது.

கீழே உள்ள எடுத்துக்காட்டில், CSS animation பண்புடன் ஒரு பொருளை எவ்வாறு நகர்த்தலாம் என்று பார்க்கிறோம்.

<template>
  <h1>Basic CSS Animation</h1>

<script>
export default {

<style scoped>

  }

    from {
      translate: 0 0;

  }

  }

  }
</style>

<Transition> கூறு

மேலே உள்ள இரண்டு எடுத்துக்காட்டுகளில் நாம் செய்தது போல பிளேன் CSS transitions மற்றும் animations ஐப் பயன்படுத்துவதில் தவறில்லை.

ஆனால் அதிர்ஷ்டவசமாக Vue v-if அல்லது v-show உடன் ஒரு உறுப்பு அகற்றப்படும் போது அல்லது சேர்க்கப்படும் போது அனிமேஷன் செய்ய விரும்பும் சந்தர்ப்பங்களில் உள்ளமைக்கப்பட்ட <Transition> கூறை வழங்குகிறது, ஏனெனில் அது பிளேன் CSS animation உடன் செய்வது கடினமாக இருக்கும்.

முதலில் ஒரு பொத்தான் <p> டேக்கைச் சேர்க்கும் அல்லது அகற்றும் ஒரு பயன்பாட்டை உருவாக்குவோம்:

<template>
  <h1>Add/Remove <p> Tag</h1>
  <button @click="this.exists = !this.exists">{{btnText}}</button><br>
  <p v-if="exists">Hello World!</p>
</template>

<script>
export default {
  data() {
    return {
      exists: false
    }
  },
  computed: {
    btnText() {
      if(this.exists) {
        return 'Remove';
      }
      else {
        return 'Add';
      }
    }
  }
}
</script>

இப்போது <Transition> கூறை <p> டேக்கைச் சுற்றி wrap செய்து, <p> டேக்கை அகற்றுவதை எவ்வாறு அனிமேட் செய்யலாம் என்று பார்ப்போம்.

நாம் <Transition> கூறைப் பயன்படுத்தும் போது, உறுப்புகள் சேர்க்கப்படும் போது அல்லது அகற்றப்படும் போது அனிமேஷன் செய்ய பயன்படுத்தக்கூடிய ஆறு வெவ்வேறு CSS வகுப்புகளை தானாகவே பெறுகிறோம்.

ஆறு <Transition> வகுப்புகள்

நாம் <Transition> கூறைப் பயன்படுத்தும் போது நமக்குத் தானாகவே கிடைக்கும் ஆறு வகுப்புகள் உள்ளன.

<Transition> கூறுக்குள் ஒரு உறுப்பு சேர்க்கப்படும் போது, அந்த transition ஐ அனிமேட் செய்ய இந்த முதல் மூன்று வகுப்புகளைப் பயன்படுத்தலாம்:

<Transition> கூறின் ரூட் நிலையில் ஒரே ஒரு உறுப்பு மட்டுமே இருக்க முடியும்.

இப்போது, <p> டேக் சேர்க்கப்படும் போதும், அகற்றப்படும் போதும் அனிமேட் செய்ய இந்த வகுப்புகளில் நான்கைப் பயன்படுத்துவோம்.

✗ தவறு! Vue Transition வகுப்புகள் 'enter' மற்றும் 'leave' என்று பெயரிடப்பட்டுள்ளன, 'start', 'during', 'end' அல்ல

உறுப்புகளுக்கு இடையே Transition

<Transition> கூறு பல உறுப்புகளுக்கு இடையில் மாற்றவும் பயன்படுத்தப்படலாம், <v-if> மற்றும் <v-else-if> பயன்படுத்தி ஒரே நேரத்தில் ஒரே ஒரு உறுப்பு மட்டுமே காட்டப்படுவதை உறுதி செய்தால்:

<template>
  <h1>Transition Between Elements</h1>
  <button @click="newImg">Next image</button><br>
  <Transition>
    <img src="/img_pizza.svg" v-if="imgActive === 'pizza'">
    <img src="/img_apple.svg" v-else-if="imgActive === 'apple'">
    <img src="/img_cake.svg" v-else-if="imgActive === 'cake'">
    <img src="/img_fish.svg" v-else-if="imgActive === 'fish'">
    <img src="/img_rice.svg" v-else-if="imgActive === 'rice'">
  </Transition>
</template>

mode="out-in"

மேலே உள்ள எடுத்துக்காட்டில், முந்தைய படம் அகற்றப்படுவதற்கு முன்பே அடுத்த படம் சேர்க்கப்படுகிறது.

ஒரு உறுப்பின் நீக்குதல் முடிவடைவதற்கு முன்பு அடுத்த உறுப்பு சேர்க்கப்படாமல் இருப்பதற்கு <Transition> கூறில் mode="out-in" prop மற்றும் prop மதிப்பைப் பயன்படுத்துகிறோம்.

<Transition mode="out-in">
  <img src="/img_pizza.svg" v-if="imgActive === 'pizza'">
  <img src="/img_apple.svg" v-else-if="imgActive === 'apple'">
  <img src="/img_cake.svg" v-else-if="imgActive === 'cake'">
</Transition>

💡 mode விருப்பங்கள்:

mode="out-in": முந்தைய உறுப்பு முழுமையாக வெளியேறிய பிறகு புதிய உறுப்பு நுழைகிறது
mode="in-out": புதிய உறுப்பு முதலில் நுழைகிறது, பின்னர் முந்தைய உறுப்பு வெளியேறுகிறது
இல்லை mode: இரண்டும் ஒரே நேரத்தில் நடக்கும் (இயல்புநிலை)

டைனமிக் கூறுகளுடன் Transition

டைனமிக் கூறுகளுக்கு இடையில் மாறுவதை அனிமேட் செய்ய <Transition> கூறைப் பயன்படுத்தலாம்:

<template>
  <h1>Transition with Dynamic Components</h1>
  <button @click="toggleValue = !toggleValue">Switch component</button>
  <Transition mode="out-in">
    <component :is="activeComp"></component>
  </Transition>
</template>

<script>
  export default {
    data () {
      return {
        toggleValue: true
      }
    },
    computed: {
      activeComp() {
        if(this.toggleValue) {
          return 'comp-one'
        }
        else {
          return 'comp-two'
        }
      }
    }
  }
</script>

CSS transitions பயன்படுத்தவும்: எளிமையான அனிமேஷன்களுக்கு CSS transitions பயன்படுத்தவும், அவை மிகவும் செயல்திறன் வாய்ந்தவை
hardware acceleration பயன்படுத்தவும்:
அனிமேஷன் கால அளவுகளை கவனிக்கவும்: 100-300ms க்கு இடையில் வைக்கவும், மிக நீண்ட அனிமேஷன்கள் பயனர் பொறுமையை இழக்கச் செய்யும்
layout-changing properties தவிர்க்கவும்: top, left, width, height போன்ற பண்புகள் மறுஅளவிடுதலைத் தூண்டும்
easing செயல்பாடுகளைப் பயன்படுத்தவும்: ease-in-out, cubic-bezier() போன்ற easing செயல்பாடுகள் இயற்கையான இயக்கத்தை உருவாக்குகின்றன
<!-- Good: Hardware accelerated -->
.v-enter-active {
  animation: slideIn 0.3s ease-out;
  transform: translateX(0);
  opacity: 1;
}

@keyframes slideIn {
  from {
    transform: translateX(-100px);
    opacity: 0;
  }
}

<!-- Avoid: Causes layout reflow -->
.v-enter-active {
  animation: slideIn 0.3s;
  left: 0;
}

@keyframes slideIn {
  from {
    left: -100px;
  }
}
