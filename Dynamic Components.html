Dynamic Components

உங்கள் பக்கத்தில் பக்கங்களைத் திருப்ப, உங்கள் பிரௌசரில் உள்ள தாவல்களைப் போல, 'is' பண்புக்கூறைப் பயன்படுத்தி Dynamic Components பயன்படுத்தப்படலாம்.

'is' பண்புக்கூறு மாறுகிறது

ஒரு டைனமிக் கூறை உருவாக்க, செயலில் உள்ள கூறைக் குறிக்க <component> டேக்கைப் பயன்படுத்துகிறோம். 'is' பண்புக்கூறு v-bind உடன் ஒரு மதிப்புடன் இணைக்கப்பட்டுள்ளது, மேலும் நாம் செயலில் இருக்க விரும்பும் கூறின் பெயருக்கு அந்த மதிப்பை மாற்றுகிறோம்.

எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

// App.vue:
<template>
  <h1>Dynamic Components</h1>
  <p>App.vue switches between which component to show.</p>
  <button @click="toggleValue = !toggleValue">
    Switch component
  </button>
  <component :is="activeComp"></component>
</template>

<script>
  export default {
    data() {
      return {
        toggleValue: true
      }
    },
    computed: {
      activeComp() {
        if(this.toggleValue) {
          return 'comp-one'
        }
        else {
          return 'comp-two'
        }
      }
    }
  }
</script>

🔄 விளக்கம்:

இந்த எடுத்துக்காட்டில், comp-one கூறு அல்லது comp-two கூறிற்கான பிடிப்பானாக செயல்படும் ஒரு <component> டேக் உள்ளது. 'is' பண்புக்கூறு <component> டேக்கில் அமைக்கப்பட்டு, 'comp-one' அல்லது 'comp-two' ஐ மதிப்பாக வைத்திருக்கும் கம்ப்யூடட் மதிப்பான 'activeComp' ஐக் கேட்கிறது. மேலும் கம்ப்யூடட் மதிப்பு செயலில் உள்ள கூறுகளுக்கு இடையே மாறுவதற்கு 'true' மற்றும் 'false' இடையே ஒரு தரவுப் பண்பை மாற்றும் ஒரு பொத்தான் உள்ளது.

சிக்கல்: நிலை இழப்பு

கீழே உள்ள எடுத்துக்காட்டை இயக்கவும். ஒரு கூறில் நீங்கள் செய்யும் மாற்றங்கள், நீங்கள் அதற்குத் திரும்பும் போது மறக்கப்பட்டுவிடும் என்பதை நீங்கள் கவனிப்பீர்கள். ஏனெனில் கூறு unmount செய்யப்பட்டு மீண்டும் mount செய்யப்படுகிறது, கூறு மீண்டும் ஏற்றப்படுகிறது.

<KeepAlive> தீர்வு

கூறுக்குத் திரும்பும் போது, உங்கள் முந்தைய உள்ளீடுகள், நிலையை வைத்திருக்க, <component> டேக்கைச் சுற்றி <KeepAlive> டேக்கைப் பயன்படுத்துகிறோம்.

எடுத்துக்காட்டு

கூறுகள் இப்போது பயனர் உள்ளீடுகளை நினைவில் வைக்கின்றன.

// App.vue:
<template>
  <h1>Dynamic Components</h1>
  <p>App.vue switches between which component to show.</p>
  <button @click="toggleValue = !toggleValue">
    Switch component
  </button>
  <KeepAlive>
    <component :is="activeComp"></component>
  </KeepAlive>
</template>

'include' மற்றும் 'exclude' பண்புக்கூறுகள்

<KeepAlive> டேக்கிற்குள் உள்ள அனைத்து கூறுகளும் இயல்பாக வைத்திருக்கப்படும்.

ஆனால் <KeepAlive> டேக்கில் 'include' அல்லது 'exclude' பண்புக்கூறுகளைப் பயன்படுத்தி சில கூறுகள் மட்டுமே வைத்திருக்கப்படும் என வரையறுக்கலாம்.

<KeepAlive> டேக்கில் 'include' அல்லது 'exclude' பண்புக்கூறுகளைப் பயன்படுத்தினால், 'name' விருப்பத்துடன் கூறுகளுக்குப் பெயர்களையும் கொடுக்க வேண்டும்:

// CompOne.vue:
<script>
  export default {
    name: 'CompOne',
    data() {
      return {
        imgSrc: 'img_question.svg'
      }
    }
  }
</script>

include

வைத்திருக்க வேண்டிய கூறுகளைக் குறிப்பிடுகிறது

<KeepAlive include="CompOne">

exclude

வைத்திருக்கக்கூடாத கூறுகளைக் குறிப்பிடுகிறது

<KeepAlive exclude="CompOne">

பல கூறுகள்

கமா பிரிப்புடன் பல கூறுகள்

<KeepAlive include="CompOne,CompThree">

include/exclude எடுத்துக்காட்டுகள்

include எடுத்துக்காட்டு

// App.vue:
<template>
  <h1>Dynamic Components</h1>
  <p>App.vue switches between which component to show.</p>
  <button @click="toggleValue = !toggleValue">
    Switch component
  </button>
  <KeepAlive include="CompOne">
    <component :is="activeComp"></component>
  </KeepAlive>
</template>

<KeepAlive include="CompOne"> உடன், 'CompOne' கூறு மட்டுமே அதன் நிலையை, முந்தைய உள்ளீடுகளை நினைவில் வைக்கும்.

exclude எடுத்துக்காட்டு

// App.vue:
<template>
  <h1>Dynamic Components</h1>
  <p>App.vue switches between which component to show.</p>
  <button @click="toggleValue = !toggleValue">
    Switch component
  </button>
  <KeepAlive exclude="CompOne">
    <component :is="activeComp"></component>
  </KeepAlive>
</template>

<KeepAlive exclude="CompOne"> உடன், 'CompTwo' கூறு மட்டுமே அதன் நிலையை நினைவில் வைக்கும்.

'max' பண்புக்கூறு

பிரௌசர் நிலையை நினைவில் வைத்திருக்க வேண்டிய கூறுகளின் எண்ணிக்கையைக் கட்டுப்படுத்த <KeepAlive> டேக்கில் ஒரு பண்புக்கூறாக 'max' ஐப் பயன்படுத்தலாம்.

எடுத்துக்காட்டு

// App.vue:
<template>
  <h1>Dynamic Components</h1>
  <label><input type="radio" name="rbgComp" v-model="compName" :value="'comp-one'"> One</label>
  <label><input type="radio" name="rbgComp" v-model="compName" :value="'comp-two'"> Two</label>
  <label><input type="radio" name="rbgComp" v-model="compName" :value="'comp-three'"> Three</label>
  <KeepAlive :max="2">
    <component :is="activeComp"></component>
  </KeepAlive>
</template>

📊 மேம்பாடு:

<KeepAlive :max="2"> உடன், பிரௌசர் கடைசியாகப் பார்வையிட்ட இரண்டு கூறுகளின் பயனர் உள்ளீட்டை மட்டுமே நினைவில் வைக்கும். இது நினைவகப் பயன்பாட்டை மேம்படுத்துகிறது.
