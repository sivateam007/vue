Vue Forms

Vue பயனர் அனுபவத்தை மேம்படுத்த ஒரு எளிதான வழியை வழங்குகிறது, பதிலளிப்பு மற்றும் படிவ மதிப்பாய்வு போன்ற கூடுதல் செயல்பாடுகளைச் சேர்ப்பதன் மூலம்.

Vue படிவங்களைக் கையாளும்போது v-model directive ஐப் பயன்படுத்துகிறது.

Our First Form with Vue

Vue படிவத்தை உருவாக்கும் போது எவ்வாறு பயன்படுத்தப்படலாம் என்பதைப் பார்க்க ஒரு எளிய ஷாப்பிங் பட்டியல் உதாரணத்துடன் தொடங்குவோம்.

HTML இல் படிவங்கள் பற்றிய மேலும் தகவலுக்கு, தொடர்புடைய குறிச்சொற்கள் மற்றும் பண்புக்கூறுகளுடன், எங்கள் HTML படிவங்கள் டுடோரியலைப் பார்க்கவும்.

📝 படி 1:

நிலையான HTML படிவ கூறுகளைச் சேர்க்கவும்:

<form>
  <p>Add item</p>
  <p>Item name: <input type="text" required></p>
  <p>How many: <input type="number"></p>
  <button type="submit">Add item</button>
</form>

📝 படி 2:

தற்போதைய உருப்படி பெயர், எண் மற்றும் ஷாப்பிங் பட்டியலுடன் Vue உதாரணத்தை உருவாக்கி, எங்கள் உள்ளீடுகளை இணைக்க v-model ஐப் பயன்படுத்தவும்.

<div id="app">
  <form>
    <p>Add item</p>
    <p>Item name: <input type="text" required v-model="itemName"></p>
    <p>How many: <input type="number" v-model="itemNumber"></p>
    <button type="submit">Add item</button>
  </form>
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        itemName: null,
        itemNumber: null,
        shoppingList: [
          { name: 'Tomatoes', number: 5 }
        ]
      }
    }
  })
  app.mount('#app')
</script>

📝 படி 3:

ஷாப்பிங் பட்டியலில் ஒரு உருப்படியைச் சேர்க்க முறையை அழைக்கவும், மற்றும் சமர்ப்பிப்பில் இயல்புநிலை உலாவி புதுப்பிப்பைத் தடுக்கவும்.

<form v-on:submit.prevent="addItem">

📝 படி 4:

ஷாப்பிங் பட்டியலில் உருப்படியைச் சேர்க்கும் முறையை உருவாக்கவும், மற்றும் படிவத்தைத் துடைக்கவும்:

methods: {
  addItem() {
    let item = {
      name: this.itemName,
      number: this.itemNumber
      }
    this.shoppingList.push(item);
    this.itemName = null
    this.itemNumber = null
  }
}

📝 படி 5:

படிவத்திற்கு கீழே தானாக புதுப்பிக்கப்பட்ட ஷாப்பிங் பட்டியலைக் காட்ட v-for ஐப் பயன்படுத்தவும்:

<p>Shopping list:</p>
<ul>
  <li v-for="item in shoppingList">{{item.name}}, {{item.number}}</li>
</ul>

Example

இந்த உதாரணத்தில் நாம் ஒரு ஷாப்பிங் பட்டியலுக்கு புதிய உருப்படிகளைச் சேர்க்கலாம்.

<div id="app">
  <form v-on:submit.prevent="addItem">
    <p>Add item</p>
    <p>Item name: <input type="text" required v-model="itemName"></p>
    <p>How many: <input type="number" v-model="itemNumber"></p>
    <button type="submit">Add item</button>
  </form>

  <p>Shopping list:</p>
  <ul>
    <li v-for="item in shoppingList">{{item.name}}, {{item.number}}</li>
  </ul>
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        itemName: null,
        itemNumber: null,
        shoppingList: [
          { name: 'Tomatoes', number: 5 }
        ]
      }
    },
    methods: {
      addItem() {
        let item = {
          name: this.itemName,
          number: this.itemNumber
        }
        this.shoppingList.push(item)
        this.itemName = null
        this.itemNumber = null
      }
    }
  })
  app.mount('#app')
</script>

மேலே உள்ள உதாரணத்தில் v-model வழங்கும் இரு-வழி பிணைப்பைக் கவனியுங்கள்:

  • HTML உள்ளீடு மாறும்போது v-model Vue உதாரண தரவைப் புதுப்பிக்கிறது
  • Vue உதாரண தரவு மாறும்போது v-model HTML உள்ளீட்டைப் புதுப்பிக்கிறது

v-model பற்றி மேலும் அறிய மற்றும் மேலும் படிவ உதாரணங்களைப் பார்க்க, 'அடுத்தது' என்பதைக் கிளிக் செய்யவும்.

Vue பயிற்சிகள்

Vue படிவங்களில் உங்கள் அறிவைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue.js இல் படிவ சமர்ப்பிப்பைக் கையாள எந்த நிகழ்வு மாற்றி பயன்படுத்தப்படுகிறது?

v-on:submit.stop
✗ தவறு! .stop மாற்றி நிகழ்வு பரப்பலை நிறுத்துகிறது
v-on:submit.prevent
✓ சரி! .prevent மாற்றி பிரௌசரின் இயல்புநிலை செயலைத் தடுக்கிறது
v-on:submit.capture
✗ தவறு! .capture மாற்றி பிடிப்பு பயன்முறையில் நிகழ்வுகளைக் கையாளுகிறது
v-on:submit.self
✗ தவறு! .self மாற்றி உறுப்பு கிளிக்கப்பட்டால் மட்டுமே நிகழ்வுகளைக் கையாளுகிறது
