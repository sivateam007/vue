உங்கள் Vue ப்ராஜெக்ட்டை பில்ட் செய்வது கற்றுக்கொள்ளுங்கள்

உங்கள் ப்ராஜெக்ட்டை பில்ட் செய்யுங்கள்

ஒரு Vue ப்ராஜெக்ட் முடிந்ததும், அது "டெவலப்மென்ட் மோட்" இலிருந்து "பில்ட்" மோட்டுக்கு நகர வேண்டும்.

பில்ட் கட்டளை எங்கள் Vue ப்ராஜெக்ட்டை .html, .js மற்றும் .css கோப்புகளாக தொகுக்கிறது, அவை பிரௌசரில் நேரடியாக இயக்குவதற்கு ஓப்டிமைஸ் செய்யப்பட்டவை.

மற்றவர்கள் அணுகுவதற்கு ஒரு சர்வரில் கோப்புகளை உருவாக்க எங்கள் Vue ப்ராஜெக்ட்டை நாங்கள் பில்ட் செய்கிறோம்.

வலைப் பக்கத்தை 'பில்ட்' செய்ய

இந்த டுடோரியலில் இதுவரை, எங்கள் ப்ராஜெக்ட்கள் டெவலப்மென்ட் மோட்டில் இயங்கிக்கொண்டிருக்கின்றன, அதாவது Vite பில்ட் கருவி ஒரு டெவலப்மென்ட் சேவையகத்தை இயக்குகிறது. நீங்கள் டெவலப்மென்டின் போது மாற்றங்களைச் செய்து அவற்றைச் சேமிக்கும்போது, Vite உடனடியாக பக்கத்தைப் புதுப்பிக்கும். இதற்கு கணினியில் இருந்து நிறைய வளங்கள் தேவை.

பக்கத்தை பொது மக்களுக்கு வெளியிட தயாராக இருக்கும் போது, டெவலப்மென்ட் கட்டத்திற்குப் பிறகு பில்ட் படி வருகிறது. பின்னர் Vite ஐ டெவலப்மென்ட் மோட்டில் இயக்காமல், எங்கள் ப்ராஜெக்ட்டை பிரௌசர் புரிந்துகொள்ளும் கோப்புகளாக பில்ட் செய்ய வேண்டும். சர்வர் வள பயன்பாட்டைக் குறைக்கவும் செயல்திறனை மேம்படுத்தவும் பில்ட் படி செய்யப்படுகிறது.

உங்கள் Vue பயன்பாட்டை பில்ட் செய்ய, டெவலப்மென்ட் சேவையகம் இயங்கிக்கொண்டிருந்தால், 'Q' அல்லது 'ctrl'+'C' ஐ அழுத்துவதன் மூலம் நிறுத்தவும், பின்னர் எழுதவும்:

npm run build

உங்கள் ப்ராஜெக்ட் பில்ட் செய்யப்படும்போது, Vite ஒரு dist என்ற கோப்புறையை உருவாக்குகிறது, பொது சேவையகத்தில் உங்கள் ப்ராஜெக்ட்டை இயக்க தேவையான அனைத்து கோப்புகளுடன், பிரௌசர் புரிந்துகொள்ளும் *.html, *.css மற்றும் *.js கோப்புகளுடன், டெவலப்மென்டின் போது நாம் பயன்படுத்தும் *.vue கோப்புகளுக்கு பதிலாக.

📁 படம் 30: Dist கோப்புறை

Vue ப்ராஜெக்ட் பில்ட் செய்த பிறகு Dist கோப்புறை அமைப்பு

பிரௌசரில் உங்கள் பில்ட் செய்யப்பட்ட ப்ராஜெக்ட்டைப் பார்க்க, கட்டளையைப் பயன்படுத்தவும்:

npm run preview

இந்த கட்டளை dist கோப்புறையில் இருந்து பில்ட் செய்யப்பட்ட ப்ராஜெக்ட்டைக் காட்டும் ஒரு பிரௌசர் விண்டோவைத் திறக்க வேண்டும்.
