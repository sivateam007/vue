Vue $el Object

Vue $el பொருளைப் பற்றி அறிந்து கொள்ளுங்கள்

Vue $el பொருள்

$el பொருள் Vue கூறின் மூல DOM முனையைக் குறிக்கிறது.

Vue பயன்பாடு ஏற்றப்படும் வரை $el பொருள் இருக்காது.

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

ரூட் நிலையில் <div> குறியின் பின்புப்பக்க வண்ணத்தை மாற்ற $el பொருளைப் பயன்படுத்துதல்.

Example

methods: {
  changeColor() {
    this.$el.style.backgroundColor = 'lightgreen';
  }
}

💡 குறிப்பு:

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

$el பொருள் Vue கூறின் மூல DOM முனையைக் குறிக்கிறது.

Vue பயன்பாடு ஏற்றப்படும் வரை $el பொருள் இருக்காது.

<template> இல் ஒரே ஒரு HTML மூல உறுப்பு மட்டும் இருந்தால்:

  • $el பொருள் அந்த மூல உறுப்பாக இருக்கும்.
  • $el பொருளைப் பயன்படுத்தி நேரடியாக DOM ஐ மாற்றியமைக்க முடியும் (மேலே உள்ள எடுத்துக்காட்டைப் பார்க்கவும்), ஆனால் இது பரிந்துரைக்கப்படவில்லை.
  • மிகவும் நிலையான மற்றும் பராமரிக்க எளிதான குறியீட்டிற்கு, DOM ஐ மாற்ற Vue ref பண்பு மற்றும் பிற Vue செயல்பாடுகளைப் பயன்படுத்துவது நல்லது (கீழே உள்ள எடுத்துக்காட்டு 1ஐப் பார்க்கவும்).

<template> இல் ஒன்றுக்கு மேற்பட்ட HTML மூல உறுப்புகள் இருந்தால்:

  • $el பொருள் Vue உள் பயன்பாட்டிற்கான ஒரு placeholder DOM உரை முனையாக இருக்கும் (உண்மையான DOM உறுப்பு அல்ல).
  • பல மூல உறுப்புகள் இருக்கும்போது $el பொருளைப் பயன்படுத்தி DOM ஐ மாற்றியமைக்க முடியாது (கீழே உள்ள எடுத்துக்காட்டு 2ஐப் பார்க்கவும்).

⚠️ கவனிக்க:

Vue 3 இன் Composition API இல், $el பண்பு <script setup> இல் அணுக முடியாது (setup செயல்பாட்டைப் பயன்படுத்தி).

மேலும் எடுத்துக்காட்டுகள்

Example 1

பரிந்துரைக்கப்பட்டபடி, $el பொருளைப் பயன்படுத்துவதற்குப் பதிலாக ref பண்பைப் பயன்படுத்தி <div> குறியின் பின்புப்பக்க வண்ணத்தை மாற்றுவது.

Template

<template>
  <div ref="rootDiv">
    <h2>Example $el Object</h2>
    <p>It is recommended, and more consistent, to use the ref attribute instead of the $el object to change the background color root DIV tag.</p>
    <button v-on:click="changeColor">Click here</button>
  </div>
</template>

Script

<script>
export default {
  methods: {
    changeColor() {
      this.$refs.rootDiv.style.backgroundColor = 'lightgreen';
    }
  }
}
</script>

Example 2

<template> இன் மூலத்தில் ஒன்றுக்கு மேற்பட்ட உறுப்புகள் இருக்கும்போது, $el பொருள் Vue மூலம் உள் பயன்பாட்டிற்கான ஒரு உரை முனை பிரதிநிதித்துவமாக இருக்கும் (உண்மையான DOM உறுப்பு அல்ல).

இதுபோன்ற சந்தர்ப்பங்களில் $el பொருளுடன் DOM ஐ மாற்றியமைக்க முடியாது.

Template

<template>
  <div>
    <h2>Example $el Object</h2>
    <p>We are not able to use the $el object to change the background color of the root DIV tag when there are other tags on the root level. Open browser console to see the error generated.</p>
    <button v-on:click="changeColor">Click here</button>
  </div>
  <p>With this extra p-tag there are two tags on the root level.</p>
</template>

Example 3

<h2> குழந்தை உறுப்பின் பின்புப்பக்க வண்ணத்தை மாற்ற $el பொருளைப் பயன்படுத்துவது.

Script

<script>
export default {
  methods: {
    changeColor() {
      this.$el.children[0].style.backgroundColor = 'lightblue';
    }
  }
}
</script>

பயிற்சி

❓ Vue $el பொருள் எப்போது உருவாக்கப்படுகிறது?

Vue கூறு உருவாக்கப்பட்ட உடனேயே
✗ தவறு! Vue கூறு உருவாக்கப்பட்ட உடனேயே $el பொருள் உருவாக்கப்படாது
தொகுதி உருவாக்கப்பட்ட உடனேயே
✗ தவறு! தொகுதி உருவாக்கப்பட்ட உடனேயே $el பொருள் கிடைக்காது
Vue பயன்பாடு ஏற்றப்பட்ட பிறகு
✓ சரி! Vue பயன்பாடு ஏற்றப்பட்ட பிறகு மட்டுமே $el பொருள் கிடைக்கும்
கட்டளை வரியில் உள்ளபோது
✗ தவறு! கட்டளை வரியில் இருப்பது $el உருவாக்கத்தை பாதிக்காது
