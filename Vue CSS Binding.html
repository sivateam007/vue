Vue CSS Binding

Vue CSS பைண்டிங்கைக் கற்றுக்கொள்ளுங்கள்

Vue CSS Binding

style மற்றும் class பண்புக்கூறுகளுடன் CSS ஐ மாற்ற v-bind ஐ எவ்வாறு பயன்படுத்துவது என்பதைப் பற்றி மேலும் அறிக.

v-bind உடன் style மற்றும் class பண்புக்கூறுகளை மாற்றுவதற்கான கருத்து மிகவும் நேரடியானது, ஆனால் தொடரியல் சில பழக்கத்தைத் தேவைப்படுத்தலாம்.

Dynamic CSS in Vue

style மற்றும் class பண்புக்கூறுகளில் v-bind ஐப் பயன்படுத்துவதன் மூலம் CSS ஐ மாற்ற Vue ஐ எவ்வாறு பயன்படுத்தலாம் என்பதை நீங்கள் ஏற்கனவே பார்த்திருக்கிறீர்கள். இது v-bind கீழ் இந்த டுடோரியலில் சுருக்கமாக விளக்கப்பட்டுள்ளது மற்றும் Vue CSS ஐ மாற்றும் பல எடுத்துக்காட்டுகளும் கொடுக்கப்பட்டுள்ளன.

இங்கே Vue உடன் CSS எவ்வாறு டைனமிகலாக மாற்றப்படலாம் என்பதை மேலும் விரிவாக விளக்குவோம். ஆனால் முதலில் இந்த டுடோரியலில் நாம் ஏற்கனவே பார்த்த இரண்டு உதாரணங்களைப் பார்ப்போம்: v-bind:style உடன் இன்-லைன் ஸ்டைலிங் மற்றும் v-bind:class உடன் ஒரு வகுப்பை ஒதுக்குதல்

Inline Styling

Vue இல் இன்-லைன் ஸ்டைலிங் செய்ய v-bind:style ஐப் பயன்படுத்துகிறோம்.

🎯 Example

ஒரு <div> உறுப்பின் ஒளிபுகா தன்மையை மாற்ற இன்-லைன் ஸ்டைலிங் பயன்படுத்தி ஒரு <input type="range"> உறுப்பு பயன்படுத்தப்படுகிறது.

<input type="range" v-model="opacityVal">
<div v-bind:style="{ backgroundColor: 'rgba(155,20,20,'+opacityVal+')' }">
  Drag the range input above to change opacity here.
</div>

Assign a Class

Vue இல் ஒரு HTML குறிச்சொல்லுக்கு ஒரு வகுப்பை ஒதுக்க v-bind:class ஐப் பயன்படுத்துகிறோம்.

🎯 Example

உணவின் படங்களைத் தேர்ந்தெடுக்கவும். நீங்கள் தேர்ந்தெடுத்ததைக் காட்டுவதற்கு v-bind:class ஐப் பயன்படுத்தி தேர்ந்தெடுக்கப்பட்ட உணவு முன்னிலைப்படுத்தப்படுகிறது.

<div v-for="(img, index) in images">
  <img v-bind:src="img.url"
       v-on:click="select(index)"
       v-bind:class="{ selClass: img.sel }">
</div>

Other Ways to Assign Classes and Style

v-bind:class மற்றும் v-bind:style பயன்பாடு தொடர்பான வெவ்வேறு அம்சங்கள் இங்கே உள்ளன, அவை இந்த டுடோரியலில் நாம் முன்பு பார்த்ததில்லை:

  • class="" மற்றும் v-bind:class="" ஆகிய இரண்டிலும் CSS வகுப்புகள் ஒரு HTML குறிச்சொல்லுக்கு ஒதுக்கப்படும்போது Vue வகுப்புகளை இணைக்கிறது.
  • v-bind:class="{}" உடன் ஒன்று அல்லது அதற்கு மேற்பட்ட வகுப்புகளைக் கொண்ட ஒரு பொருள் ஒதுக்கப்படுகிறது. பொருளுக்குள் ஒன்று அல்லது அதற்கு மேற்பட்ட வகுப்புகள் இயக்கப்படலாம் அல்லது முடக்கப்படலாம்.
  • இன்-லைன் ஸ்டைலிங் (v-bind:style) உடன் ஒரு CSS பண்புக்கூறை வரையறுக்கும்போது கேமல் கேஸ் விரும்பப்படுகிறது, ஆனால் அது மேற்கோள்களுக்குள் எழுதப்பட்டால் 'கேபப்-கேஸ்'யும் பயன்படுத்தலாம்.
  • CSS வகுப்புகளை வரிசைகள் / வரிசை குறியீட்டு முறை / தொடரியல் மூலம் ஒதுக்க முடியும்

இந்த புள்ளிகள் கீழே மேலும் விரிவாக விளக்கப்பட்டுள்ளன.

1. Vue Merges 'class' And 'v-bind:class'

class="" மூலம் ஒதுக்கப்பட்ட வகுப்பிற்கு ஒரு HTML குறிச்சொல் சொந்தமானது மற்றும் v-bind:class="" மூலம் ஒரு வகுப்பிற்கும் ஒதுக்கப்படும் போது, Vue எங்களுக்கு வகுப்புகளை இணைக்கிறது.

🎯 Example

ஒரு <div> உறுப்பு இரண்டு வகுப்புகளுக்குச் சொந்தமானது: 'impClass' மற்றும் 'yelClass'. 'important' வகுப்பு வகுப்பு பண்புக்கூறுடன் சாதாரண வழியில் அமைக்கப்பட்டுள்ளது, மற்றும் 'yellow' வகுப்பு v-bind:class உடன் அமைக்கப்பட்டுள்ளது.

<div class="impClass" v-bind:class="{yelClass: isYellow}">
  This div belongs to both 'impClass' and 'yelClass'.
</div>

2. Assign More Than One Class With 'v-bind:class'

v-bind:class="{}" உடன் ஒரு HTML உறுப்பை ஒரு வகுப்பிற்கு ஒதுக்கும்போது, பல வகுப்புகளைப் பிரிக்கவும் ஒதுக்கவும் வெறுமனே கமாவைப் பயன்படுத்தலாம்.

🎯 Example

ஒரு <div> உறுப்பு பூலியன் Vue தரவு பண்புகள் 'isYellow' மற்றும் 'isImportant' ஆகியவற்றைப் பொறுத்து 'impClass' மற்றும் 'yelClass' வகுப்புகளுக்குச் சொந்தமானது.

<div v-bind:class="{yelClass: isYellow, impClass: isImportant}">
  This tag can belong to both the 'impClass' and 'yelClass' classes.
</div>

3. Camel case vs kebab case notation with 'v-bind:style'

இன்-லைன் ஸ்டைலிங் (v-bind:style) உடன் Vue இல் CSS ஐ மாற்றும்போது, CSS பண்புக்கூறுக்கு கேமல் கேஸ் குறியீட்டைப் பயன்படுத்த பரிந்துரைக்கப்படுகிறது, ஆனால் CSS பண்புக்கூறு மேற்கோள்களுக்குள் இருந்தால் 'கேபப்-கேஸ்'யும் பயன்படுத்தலாம்.

🎯 Example

இங்கே, ஒரு <div> உறுப்பிற்கு CSS பண்புக்கூறுகள் background-color மற்றும் font-weight ஆகியவற்றை இரண்டு வெவ்வேறு வழிகளில் அமைக்கிறோம்: பரிந்துரைக்கப்பட்ட வழியில் கேமல் கேஸ் backgroundColor, மற்றும் பரிந்துரைக்கப்படாத வழியில் மேற்கோள்களில் 'கேபப்-கேஸ்' 'font-weight'. இரண்டு மாற்றுகளும் வேலை செய்கின்றன.

<div v-bind:style="{ backgroundColor: 'lightpink', 'font-weight': 'bolder' }">
  This div tag has pink background and bold text.
</div>

'Camel case' மற்றும் 'kebab case' குறியீட்டு முறைகள் இடைவெளி அல்லது நிறுத்தற்குறிகள் இல்லாமல் தொடர் சொற்களை எழுதும் வழிகளாகும்.

  • கேமல் கேஸ் குறியீட்டு முறை என்பது முதல் சொல் ஒரு சிறிய எழுத்தில் தொடங்கி, ஒவ்வொரு சொல்லும் ஒரு பெரிய எழுத்தில் தொடங்கும் போது, 'backgroundColor' அல்லது 'camelCaseNotation' போன்றது. ஒவ்வொரு பெரிய எழுத்தும் ஒரு ஒட்டகத்தின் முதுகில் ஒரு குன்றை ஒத்திருப்பதால் இது கேமல் கேஸ் என்று அழைக்கப்படுகிறது.
  • கேபப் கேஸ் குறியீட்டு முறை என்பது சொற்கள் ஒரு கோடு - உடன் பிரிக்கப்படும்போது, 'background-color' அல்லது 'kebab-case-notation' போன்றது. கோடுகள் ஒரு 'ஷிஷ் கேபாபில்' உள்ள கோல்ப் போல் இருக்கும் என்பதால் இது கேபப் கேஸ் என்று அழைக்கப்படுகிறது.

4. Array Syntax with 'v-bind:class'

பல வகுப்புகளைச் சேர்க்க v-bind:class உடன் வரிசை தொடரியலைப் பயன்படுத்தலாம். வரிசை தொடரியலுடன் Vue பண்பைப் பொறுத்து இருக்கும் வகுப்புகளையும் Vue பண்பைச் சார்ந்து இல்லாத வகுப்புகளையும் பயன்படுத்தலாம்.

🎯 Example

இங்கே, வரிசை தொடரியலுடன் CSS வகுப்புகள் 'impClass' மற்றும் 'yelClass' ஆகியவற்றை அமைக்கிறோம். 'impClass' வகுப்பு ஒரு Vue பண்பு isImportant ஐச் சார்ந்தது மற்றும் 'yelClass' வகுப்பு எப்போதும் <div> உறுப்புடன் இணைக்கப்பட்டுள்ளது.

<div v-bind:class="[{ impClass: isImportant }, 'yelClass' ]">
  This div tag belongs to one or two classes depending on the isImportant property.
</div>

Vue Exercises

Vue CSS பைண்டிங்கில் உங்கள் அறிவைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue.js இல் ஒரு HTML உறுப்புக்கு பல CSS வகுப்புகளை ஒதுக்குவதற்கான சரியான தொடரியல் எது?

v-bind:class="impClass yelClass"
✗ தவறு! இந்த தொடரியல் சரியானது அல்ல, பல வகுப்புகள் இந்த வழியில் பிரிக்கப்பட முடியாது
v-bind:class="{ impClass: isImportant, yelClass: isYellow }"
✓ சரி! இந்த தொடரியல் பல வகுப்புகளை ஒரு பொருள் குறியீட்டு மூலம் ஒதுக்க சரியானது
v-bind:class="[impClass, yelClass]"
✗ தவறு! வரிசை தொடரியல் சரியானது, ஆனால் வகுப்பு பெயர்கள் மேற்கோள்களில் இருக்க வேண்டும் அல்லது Vue பண்புகளுடன் இணைக்கப்பட வேண்டும்
v-bind:style="{ impClass: isImportant, yelClass: isYellow }"
✗ தவறு! v-bind:style CSS வகுப்புகளுக்கு அல்ல, CSS ஸ்டைல்களுக்கு மட்டுமே
