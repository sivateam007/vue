Local Components

இதுவரை நாம் கம்போனெண்ட்களைச் சேர்க்கும் முறை அவை ஒரு project இல் உள்ள அனைத்து *.vue ஃபைல்களிலிருந்தும் அணுகக்கூடியதாக இருக்கும்.

கம்போனெண்ட்கள் உள்ளூராக மாற்றப்படலாம், அதாவது அவை ஒரு குறிப்பிட்ட *.vue ஃபைலுக்குள் மட்டுமே அணுகக்கூடியவை.

Global கம்போனெண்ட்கள்

இதுவரை main.js இல் கம்போனெண்ட்களைச் சேர்த்த விதம், அந்த project இல் உள்ள மற்ற அனைத்து *.vue ஃபைல்களின் <template> உள்ளே கம்போனெண்ட்கள் அணுகக்கூடியதாக இருக்கும்.

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

கம்போனெண்ட்கள் ஒருவருக்கொருவர் அணுகக்கூடியவை என்பதைக் காட்ட CompTwo.vue மற்றும் App.vue இரண்டிலும் CompOne.vue கம்போனெண்டைப் பயன்படுத்துகிறோம்.

main.js:

import { createApp } from 'vue'
 
import App from './App.vue'
import CompOne from './components/CompOne.vue'
import CompTwo from './components/CompTwo.vue'

const app = createApp(App)
app.component('comp-one', CompOne)
app.component('comp-two', CompTwo)
app.mount('#app')

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்க "Try it Yourself" பொத்தானைக் கிளிக் செய்யவும்.

Local கம்போனெண்ட்கள்

ஒரு கம்போனெண்டை main.js இல் சேர்ப்பதற்குப் பதிலாக நேரடியாக *.vue ஃபைலின் <script> டேக்கில் சேர்க்கலாம்.

ஒரு கம்போனெண்டை நேரடியாக *.vue ஃபைலில் சேர்த்தால், அந்த கம்போனெண்ட் அந்த ஃபைலுக்கு உள்ளூராக மட்டுமே அணுகக்கூடியதாக மாறும்.

எடுத்துக்காட்டு

CompOne.vue ஐ App.vue க்கு உள்ளூராக மாற்ற, மேலும் அங்கு மட்டுமே அணுகக்கூடியதாக மாற்ற, அதை main.js இலிருந்து நீக்குகிறோம்.

main.js:

import { createApp } from 'vue'
 
import App from './App.vue'
import CompOne from './components/CompOne.vue' 
import CompTwo from './components/CompTwo.vue'
 
const app = createApp(App)
app.component('comp-one', CompOne) 
app.component('comp-two', CompTwo)
app.mount('#app')

மேலும் CompOne.vue ஐ நேரடியாக App.vue இன் <script> டேக்கில் சேர்க்கிறோம்.

App.vue:

<template>
  <h3>Local Component</h3>
  <p>The CompOne.vue component is a local component and can only be used inside App.vue.</p>
  <comp-one />
  <comp-two />
</template>

<script> 
  import CompOne from './components/CompOne.vue';

  export default {
    components: {
      'comp-one': CompOne
    }
  }
</script>

📌 முக்கியமான குறிப்பு:

CompOne.vue கம்போனெண்ட் இப்போது App.vue இல் மட்டுமே கிடைக்கும்.

Development mode இல் பயன்பாட்டை இயக்கி CompTwo.vue இலிருந்து CompOne.vue ஐப் பயன்படுத்த முயற்சித்தால், நீங்கள் ஒரு எச்சரிக்கையைப் பெறுவீர்கள்:

27.png: Development mode warning காட்டும் படம்

Global vs Local Components ஒப்பீடு

Global கம்போனெண்ட்கள்

அனைத்து *.vue ஃபைல்களிலிருந்தும் அணுகக்கூடியது

main.js இல் register செய்யப்படுகிறது

Syntax: app.component('name', Component)

பெரிய ப்ராஜெக்ட்களில் memory usage அதிகமாக இருக்கலாம்

Local கம்போனெண்ட்கள்

குறிப்பிட்ட *.vue ஃபைலுக்கு மட்டுமே அணுகக்கூடியது

கம்போனெண்ட் ஃபைலில் import செய்யப்படுகிறது

Syntax: components: { 'name': Component }

Memory usage குறைவாக இருக்கும்

எப்போது பயன்படுத்த வேண்டும்?

Global கம்போனெண்ட்கள்: பல இடங்களில் பயன்படுத்தப்படும் கம்போனெண்ட்கள் (நாவிக்கேஷன், புட்டன்கள், ஃபார்ம்கள்)
Local கம்போனெண்ட்கள்: ஒரு குறிப்பிட்ட கம்போனெண்டுக்கு மட்டுமே தேவையான கம்போனெண்ட்கள்
Local கம்போனெண்ட்கள்: தனிப்பட்ட, மறுபயன்பாடு அல்லாத கம்போனெண்ட்கள்
Local கம்போனெண்ட்கள்: பெரிய ப்ராஜெக்ட்களில் performance optimization
