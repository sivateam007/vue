Vue Instance Options

Vue உதாரண விருப்பங்களின் முழுமையான மேலோட்டம் கீழே உள்ள அட்டவணையில் உள்ளது:

விருப்பம் விளக்கம்
data நாம் அமைக்கும் தரவு பண்புகள் மற்றும் அவற்றின் ஆரம்ப மதிப்புகளைக் கொண்டுள்ளது
methods நாம் எழுதும் முறைகளைக் கொண்டுள்ளது
computed நாம் உருவாக்கும் கம்ப்யூட்டட் ப்ராபர்ட்டீஸைக் கொண்டுள்ளது, அவை தரவு பண்புகளாகப் பயன்படுத்தப்படுகின்றன, ஆனால் ஒரு வருவாய் மற்றும் வாதங்கள் இல்லாமல் செயல்பாடுகளைப் போல எழுதப்படுகின்றன, மற்றும் ஒரு சார்பு மாறும்போது அழைக்கப்படுகின்றன
watch நாம் உருவாக்கும் வாட்சர்களைக் கொண்டுள்ளது, அவை ஒரே பெயரில் உள்ள தரவு பண்பு மாறும்போது அழைக்கப்படும் செயல்பாடுகளைப் போன்றவை
props ஒரு கூறின் props ஐக் கொண்டுள்ளது, அவை பெற்றோர் கூறால் தனிப்பயன் பண்புக்கூறுகளாகப் பயன்படுத்தப்படுகின்றன
emits ஒரு கூறின் emits ஐக் கொண்டுள்ளது, ஒரு கூறு அதன் பெற்றோர் கூறுக்கு வெளியிடும் தனிப்பயன் நிகழ்வுகள்
expose பொது பண்புகளின் பட்டியலைக் கொண்டுள்ளது. வெளிப்படுத்தப்படாத பண்புகள், கூறுக்கு தனிப்பட்டதாக வைக்கப்படுகின்றன

Vue Exercises

Vue உதாரண விருப்பங்கள் பற்றிய உங்கள் அறிவைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ ஒரு கூறு அதன் பெற்றோர் கூறுக்கு தனிப்பயன் நிகழ்வுகளை வெளியிட பயன்படுத்தும் Vue உதாரண விருப்பம் எது?

emits
✓ சரி! 'emits' விருப்பம் ஒரு கூறு அதன் பெற்றோர் கூறுக்கு வெளியிடும் தனிப்பயன் நிகழ்வுகளை வரையறுக்கிறது
props
✗ தவறு! 'props' விருப்பம் பெற்றோரிடமிருந்து பெறப்பட்ட தரவுக்கு, குழந்தையிலிருந்து பெற்றோருக்கு நிகழ்வுகள் அனுப்ப அல்ல
methods
✗ தவறு! 'methods' விருப்பம் உள் முறைகளுக்கு, பெற்றோர் கூறுக்கு நிகழ்வுகள் வெளியிட அல்ல
watch
✗ தவறு! 'watch' விருப்பம் தரவு மாற்றங்களைக் கண்காணிக்க, பெற்றோர் கூறுக்கு நிகழ்வுகள் வெளியிட அல்ல
