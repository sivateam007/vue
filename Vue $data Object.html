Vue $data Object

Vue.js இல் கூறு தரவுப் பண்புகளைக் கையாள $data பொருளைக் கற்றுக்கொள்ளுங்கள்

Vue $data Object

எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

கிளிக் செய்யப்படும் போது ஒரு <div> உறுப்பின் நிறத்தை மாற்ற $data பொருளைப் பயன்படுத்துதல்.

export default {
  data() {
    return {
      color: 'lightgreen'
    }
  },
  methods: {
    changeColor() {
      this.$data.color = 'pink';
    }
  }
}

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

$data பொருள் Vue நிகழ்வின் தரவுப் பகுதியில் சேமிக்கப்பட்ட அனைத்து பண்புகளையும் குறிக்கிறது.

மேலே உள்ள எடுத்துக்காட்டில், color பண்பு this.$data.color உடன் குறிப்பிடப்படுகிறது, ஆனால் அதே பண்பை this.color என்று எழுதலாம், அது மிகவும் பொதுவானது.

Vue நிகழ்வின் தரவுப் பகுதியில் ஒரு பண்பைக் குறிப்பிடுவதை வெளிப்படையாகக் குறிக்க வேண்டியபோது $data பொருள் பயன்படுத்தப்படுகிறது.

சிறப்பு வழக்குகள்

"_" அல்லது "$" உடன் தொடங்கும் பெயரைக் கொண்ட தரவுப் பண்பைக் குறிப்பிட வேண்டும் போது $data பொருளைப் பயன்படுத்த வேண்டும், ஏனெனில் அத்தகைய சந்தர்ப்பங்களில் this முன்னொட்டு மட்டும் வேலை செய்யாது.

உதாரணம்

data() {
  return {
    _private: 'secret',
    $special: 'value'
  }
}

// this._private வேலை செய்யாது
// this.$special வேலை செய்யாது

// $data வழியாக மட்டுமே:
this.$data._private
this.$data.$special

மேலும் எடுத்துக்காட்டுகள்

எடுத்துக்காட்டு

கிளிக் செய்யும் ஒவ்வொரு முறையும் ஒரு <div> உறுப்பின் நிறத்தை மாற்ற $data பொருளைப் பயன்படுத்துதல்.

<div 
  v-on:click="$data.color+=100"
  v-bind:style="{backgroundColor:'hsl('+$data.color+',80%,80%)'}"
>
  <p>Click here</p>
</div>

நிலைமை 2: this உடன் (வெளிப்பாடுகளில்)

<div 
  @click="incrementCounter"
  :style="{ color: textColor }"
>
  {{ counter }}
</div>

<script>
export default {
  methods: {
    incrementCounter() {
      this.counter++;
    }
  }
}
</script>

💡 வெளிப்பாடுகளில் $data

டெம்ப்ளேட் வெளிப்பாடுகளில், நீங்கள் நேரடியாக $data ஐப் பயன்படுத்தலாம், இது தரவுப் பண்புகளை வெளிப்படையாகக் குறிக்க உதவுகிறது. இருப்பினும், பெரும்பாலான சந்தர்ப்பங்களில் this பயன்பாடு சுத்தமானது.

எப்போது $data ஐப் பயன்படுத்த வேண்டும்

சிறப்பு பெயர்கள்: "_" அல்லது "$" உடன் தொடங்கும் தரவுப் பண்புகளை அணுக
வெளிப்படைத்தன்மை: குறியீட்டைப் படிக்கும் போது தரவுப் பண்பு என்பதை வெளிப்படையாகக் காட்ட
டைனமிக் பண்புகள்: மாறி பெயர்களுடன் தரவுப் பண்புகளை அணுக
டெம்ப்ளேட் வெளிப்பாடுகள்: சிக்கலான டெம்ப்ளேட் வெளிப்பாடுகளில் தரவுப் பண்புகளைக் குறிப்பிட
சாதாரண அணுகல்: சாதாரண தரவுப் பண்புகளுக்கு (this.property பயன்படுத்தவும்)
கம்ப்யூடட் பண்புகள்: கம்ப்யூடட் பண்புகளை அணுக (this.computedProperty பயன்படுத்தவும்)

நடைமுறை குறியீட்டு எடுத்துக்காட்டுகள்

1. சிறப்பு பெயர்களுடன் தரவுப் பண்புகள்

export default {
  data() {
    return {
      // சாதாரண பண்புகள்
      username: 'John',
      age: 30,
      
      // சிறப்பு பண்புகள்
      _internalState: 'active',
      $metadata: { version: '1.0' },
      $$privateFlag: true
    }
  },
  methods: {
    updateInternalState() {
      // இது வேலை செய்யாது:
      // this._internalState = 'inactive';
      // this.$metadata.version = '2.0';
      
      // $data வழியாக மட்டுமே:
      this.$data._internalState = 'inactive';
      this.$data.$metadata.version = '2.0';
      this.$data.$$privateFlag = false;
    },
    logAllData() {
      // முழு $data பொருளையும் பார்க்க
      console.log(this.$data);
      
      // தனிப்பட்ட பண்புகளை அணுக
      console.log('Username:', this.username); // வேலை செய்கிறது
      console.log('Internal:', this.$data._internalState); // வேலை செய்கிறது
    }
  }
}

2. டைனமிக் பண்பு அணுகல்

export default {
  data() {
    return {
      user: {
        name: 'Alice',
        email: 'alice@example.com',
        age: 25
      }
    }
  },
  methods: {
    updateProperty(propName, value) {
      // மாறி பெயருடன் டைனமிக் அணுகல்
      this.$data.user[propName] = value;
      
      // இதற்கு சமம்:
      // if (propName === 'name') this.user.name = value;
      // if (propName === 'email') this.user.email = value;
      // if (propName === 'age') this.user.age = value;
    },
    getAllDataProperties() {
      // $data இல் அனைத்து பண்புகளையும் பெற
      return Object.keys(this.$data);
    }
  }
}

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களை சோதிக்கவும்

❓ Vue நிகழ்வின் தரவுப் பகுதியில் சேமிக்கப்பட்ட அனைத்து பண்புகளையும் குறிக்க எந்த கூறு நிகழ்வு பொருள் பயன்படுகிறது?

$attrs
✗ தவறு! $attrs பொருள் பாழ்விழும் பண்புக்கூறுகளைக் குறிக்கிறது, தரவுப் பண்புகளை அல்ல
$props
✗ தவறு! $props பொருள் பெறப்பட்ட புரொப்ஸ் களைக் குறிக்கிறது, தரவுப் பண்புகளை அல்ல
$data
✓ சரி! $data பொருள் Vue நிகழ்வின் தரவுப் பகுதியில் சேமிக்கப்பட்ட அனைத்து பண்புகளையும் குறிக்கிறது
$el
✗ தவறு! $el பொருள் ரூட் DOM முனையைக் குறிக்கிறது, தரவுப் பண்புகளை அல்ல
