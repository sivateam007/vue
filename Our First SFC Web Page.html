Our First SFC Web Page

புதிதாக எங்கள் முதல் SFC வலைப்பக்கத்தை உருவாக்க நாம்:

  • ஒரு புதிய சுத்தமான Vue திட்டத்தை உருவாக்குவோம்
  • 'App.vue' கோப்பில் குறியீட்டை எழுதுவோம்
  • வலைப்பக்கம் டெவலப்ப்மென்ட் போது தானாகவே எவ்வாறு புதுப்பிக்கப்படுகிறது என்பதைப் பார்ப்போம்
  • ப்ரொடக்ஷனுக்கு பக்கத்தை உருவாக்குவோம்

Create a Clean Project

இப்போது முந்தைய பக்கத்தில் நாம் உருவாக்கிய எடுத்துக்காட்டுத் திட்டத்தில் உள்ள அனைத்து உள்ளடக்கத்தையும் அகற்றி, Vue இல் எங்கள் சொந்த எளிய வலைப்பக்கத்தை உருவாக்குவோம்.

குறியீடு எழுதத் தொடங்குவதற்கு முன், <template>, <script> மற்றும் <style> குறிச்சொற்களுக்குள் உள்ள அனைத்து உள்ளடக்கத்தையும் அகற்றவும், மற்றும் 'setup' அல்லது 'scoped' போன்ற எந்தப் பண்புக்கூறுகளையும் அகற்றவும்.

உங்கள் 'App.vue' கோப்பு இப்போது இப்படி இருக்க வேண்டும்:

<script></script>

<template></template>

<style></style>

'src' கோப்புறையின் உள்ளே 'assets' மற்றும் 'components' கோப்புறைகளையும் அகற்றவும்.

'main.js' கோப்பில் அசெட்டுகள் இறக்குமதி செய்யப்படும் வரியை அகற்றவும், இதனால் 'main.js' இப்படி இருக்கும்:

📝 main.js

புதிய சுத்தமான main.js கோப்பு:

import { createApp } from 'vue'
import App from './App.vue'

createApp(App).mount('#app')

இப்போது எங்களிடம் வேலை செய்ய ஒரு வெற்றுத் திட்டம் உள்ளது.

Write code in 'App.vue'

இப்போது எங்களிடம் ஒரு சுத்தமான திட்டம் இருப்பதால், <template> குறிச்சொல்லின் உள்ளே ஒரு தலைப்பைச் சேர்க்கவும், இப்படி:

<template>
  <h1>Hello World!</h1>
</template>

<script></script>
<style></style>

'App.vue' கோப்பைச் சேமிக்கவும், டெர்மினலில் உள்ள localhost இணைப்பைப் பின்தொடர்ந்து உங்கள் உலாவிக்குச் செல்லவும். நீங்கள் முடிவைக் காண்கிறீர்களா? VS Code இல் நீங்கள் ஒரு மாற்றத்தைச் சேமிக்கும் ஒவ்வொரு முறையும் உலாவி இப்போது தானாகவே புதுப்பிக்கப்பட வேண்டும், உலாவியை கையினால் மீள் ஏற்ற வேண்டியதில்லை.

இப்போது சற்று பெரிய Vue எடுத்துக்காட்டைப் பார்ப்போம்:

🎯 Example

முழுமையான App.vue எடுத்துக்காட்டு:

<template>
  <h1>{{ message }}</h1>
</template>

<script>
export default {
  data() {
    return {
      message: 'This is some text'
    };
  }
};
</script>

<style></style>

💡 குறிப்பு:

மேலே உள்ள உதாரணத்தில், export default 'main.js' க்கு import App from './App.vue' உடன் தரவைப் பிடிக்கச் சாத்தியமாக்குகிறது, இதனால் அது 'index.html' உள்ளே உள்ள <div id="app"> குறிச்சொல்லில் ஏற்றப்பட முடியும்.

Development vs Production

Vue SFC திட்டத்தை இயக்குவதற்கான முக்கிய கட்டளைகள்:

Development Mode

உடனடி புதுப்பிப்புகள் மற்றும் டீபக் அம்சங்களுடன்

npm run dev

Production Build

ப்ரொடக்ஷனுக்கு உகந்த மற்றும் குறைக்கப்பட்ட கோப்புகள்

npm run build

Preview Production

ப்ரொடக்ஷன் கட்டமைப்பை உள்ளூரில் முன்னோட்டமிட

npm run preview

🚀 ப்ரொடக்ஷன் பக்கத்தை உருவாக்குதல்:

உங்கள் SFC வலைப்பக்கத்தை ப்ரொடக்ஷனுக்கு உருவாக்க, இந்த கட்டளைகளைப் பயன்படுத்தவும்:

# உகந்த ப்ரொடக்ஷன் கோப்புகளை உருவாக்க
npm run build

# ப்ரொடக்ஷன் கட்டமைப்பை உள்ளூரில் முன்னோட்டமிட
npm run preview

இது 'dist' கோப்புறையை உருவாக்கும், அங்கு HTML, CSS மற்றும் JavaScript கோப்புகள் உகந்த முறையில் இருக்கும்.
