Vue 'beforeMount' Lifecycle Hook

Vue beforeMount லைஃப்சைக்கிள் ஹுக் பற்றி அறிக

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

beforeMount மற்றும் mounted லைஃப்சைக்கிள் ஹுக்ஸ் பயன்படுத்தி mounted ஹுக்கு வரை கம்போனெண்டின் DOM எலிமெண்டுகள் கிடைக்கவில்லை என்பதை நிரூபிக்க.

export default {
  data() {
    return {
      refsObj1: '',
      refsObj2: ''
    }
  },
  beforeMount() {
    this.refsObj1 = this.$refs; // The $refs object is empty at this point
  },
  mounted() {
    this.refsObj2 = this.$refs;
  }
}

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்க "Try it Yourself" பொத்தானைக் கிளிக் செய்யவும்.

வரையறை மற்றும் பயன்பாடு

beforeMount லைஃப்சைக்கிள் ஹுக் கம்போனெண்ட் mounted ஆகும் முன் நடக்கும், எனவே கம்போனெண்ட் DOM இல் சேர்க்கப்படுவதற்கு சற்று முன்.

கம்போனெண்ட் இன்னும் mounted ஆகாததால், டேட்டா அல்லது கம்ப்யூடட் போன்ற கம்போனெண்ட் இன்ஸ்டன்ஸ் உள்ளே உள்ள பிராப்பர்ட்டிகளை அணுகலாம், ஆனால் கம்போனெண்டின் DOM எலிமெண்டுகளை இன்னும் mounted ஆகாததால் அணுக முடியாது.

beforeMount ஹுக்கில் $refs ஆப்ஜெக்ட் DOM எலிமெண்டுகளைக் கொண்டுள்ளது

beforeMount ஹுக் கம்போனெண்ட் DOM இல் சேர்க்கப்படுவதற்கு சற்று முன் நடக்கும்

beforeMount ஹுக்கில் DOM எலிமெண்டுகளை அணுகலாம்
