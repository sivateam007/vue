Vue Built-in Elements

Vue க்கு உள்ளமைக்கப்பட்ட உறுப்புகளின் தொகுப்பு உள்ளது.

உறுப்பு விளக்கம்
<component> ஒரு டைனமிக் கூறை உருவாக்குகிறது
<slot> குழந்தை கூறு பெற்றோரிடமிருந்து உள்ளடக்கத்தைப் பெறக்கூடிய ஒரு slot ஐ உருவாக்குகிறது
<template> தன்னை render செய்யாமல் உள்ளடக்கத்திற்கான ஒரு பிடிப்பானை உருவாக்குகிறது

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களை சோதிக்கவும்

❓ குழந்தை கூறு பெற்றோரிடமிருந்து உள்ளடக்கத்தைப் பெறக்கூடிய ஒரு பிடிப்பானை உருவாக்க எந்த உள்ளமைக்கப்பட்ட உறுப்பு பயன்படுத்தப்படுகிறது?

<component>
✗ தவறு! <component> டைனமிக் கூறுகளை உருவாக்க பயன்படுகிறது, பெற்றோரிடமிருந்து உள்ளடக்கத்தைப் பெற அல்ல
<slot>
✓ சரி! <slot> குழந்தை கூறு பெற்றோரிடமிருந்து உள்ளடக்கத்தைப் பெறக்கூடிய ஒரு பிடிப்பானாக செயல்படுகிறது
<template>
✗ தவறு! <template> உள்ளடக்கத்தை குழுவாக்க பயன்படுகிறது, ஆனால் பெற்றோரிடமிருந்து உள்ளடக்கத்தைப் பெற பிடிப்பானாக அல்ல
✗ தவறு! <content> என்பது Vue இல் உள்ளமைக்கப்பட்ட உறுப்பு அல்ல
