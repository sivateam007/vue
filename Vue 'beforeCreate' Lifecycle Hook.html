Vue 'beforeCreate' Lifecycle Hook

Vue beforeCreate லைஃப்சைக்கிள் ஹுக் பற்றி அறிக

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

beforeCreate லைஃப்சைக்கிள் ஹுக் பயன்படுத்தி alert உருவாக்க, கன்சோலில் எழுத, மற்றும் 'text' டேட்டா பிராப்பர்ட்டியை மாற்ற முயற்சிக்க.

beforeCreate(){
  console.log('beforeCreate lifecycle hook');
  alert('beforeCreate lifecycle hook');
  this.text = 'beforeCreate lifecycle hook'; // வேலை செய்யாது
}

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்க "Try it Yourself" பொத்தானைக் கிளிக் செய்யவும்.

வரையறை மற்றும் பயன்பாடு

beforeCreate லைஃப்சைக்கிள் ஹுக் என்பது ஒரு Vue கம்போனெண்டின் லைஃப்சைக்கிள் போது அழைக்கப்படும் முதல் ஹுக் ஆகும்.

beforeCreate லைஃப்சைக்கிள் ஹுக் கம்போனெண்ட் உருவாக்கப்படுவதற்கு முன் குறியீட்டை இயக்க பயன்படுகிறது.

கம்போனெண்ட் இன்னும் உருவாக்கப்படாததால், டேட்டா அல்லது கம்ப்யூடட் போன்ற கம்போனெண்ட் இன்ஸ்டன்ஸ் உள்ளே உள்ள பிராப்பர்ட்டிகளை அணுக முடியாது, மேலும் கம்போனெண்ட் DOM எலிமெண்டுகளையும் அணுக முடியாது, ஏனெனில் அவையும் இன்னும் உருவாக்கப்படவில்லை.

Vue பயிற்சிகள்

இந்த டுடோரியலில் உள்ள பல அத்தியாயங்கள் உங்கள் அறிவு நிலையைச் சரிபார்க்கக்கூடிய பயிற்சியுடன் முடிகின்றன.

❓ beforeCreate லைஃப்சைக்கிள் ஹுக் பற்றிய சரியான கூற்று எது?

beforeCreate இல் டேட்டா பிராப்பர்ட்டிகளை அணுகலாம்
✗ தவறு! beforeCreate ஹுக்கில் கம்போனெண்ட் இன்னும் உருவாக்கப்படாததால் டேட்டா பிராப்பர்ட்டிகளை அணுக முடியாது
beforeCreate என்பது Vue கம்போனெண்ட் லைஃப்சைக்கிளின் முதல் ஹுக் ஆகும்
✓ சரி! beforeCreate ஹுக் கம்போனெண்ட் உருவாக்கத்திற்கு முன் இயக்கப்படும் முதல் ஹுக் ஆகும்
beforeCreate இல் DOM எலிமெண்டுகளை அணுகலாம்
✗ தவறு! கம்போனெண்ட் இன்னும் உருவாக்கப்படாததால் DOM எலிமெண்டுகள் கூட இல்லை
beforeCreate ஹுக்கில் this.text போன்ற பிராப்பர்ட்டிகளை மாற்றலாம்
✗ தவறு! கம்போனெண்ட் இன்ஸ்டன்ஸ் இன்னும் உருவாக்கப்படாததால் பிராப்பர்ட்டிகள் அணுக முடியாது மற்றும் மாற்ற முடியாது
