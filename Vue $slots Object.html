Vue $slots Object

Vue $slots பொருளைப் பற்றி அறிந்து கொள்ளுங்கள்

Vue $slots பொருள்

$slots பொருள் பெற்றோரிடமிருந்து அனுப்பப்பட்ட slots ஐக் குறிக்கிறது.

பெற்றோரிடமிருந்து அனுப்பப்பட்ட slots ஐ this.$slots என்று அழைப்பதன் மூலம் பார்க்கலாம், அல்லது 'topSlot' என்று பெயரிடப்பட்ட ஒரு குறிப்பிட்ட slot ஐப் பார்க்க this.$slots.topSlot என்று அழைக்கலாம்.

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

'topSlot' க்கு பெற்றோர் உள்ளடக்கத்தை வழங்கியுள்ளாரா என்பதை சரிபார்க்க $slots பொருளைப் பயன்படுத்துவது.

Example

mounted(){
  if(this.$slots.topSlot){
    this.slotsText = "Content for the 'topSlot' slot is provided by the parent."
  }
  else {
    this.slotsText = "Content for the 'topSlot' slot is NOT provided by the parent."
  }
}
