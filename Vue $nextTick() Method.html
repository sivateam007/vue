Vue $nextTick() Method

Vue $nextTick() மெத்தட் பற்றி அறிக

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

$nextTick() மெத்தட் பயன்படுத்தி <p> டேக்கின் உள்ளே உள்ள செய்தியைப் பெறுவதற்கு முன் DOM update ஆக காத்திருக்க.

methods: {
  updateMsg() {
    this.message = '"Hello! This is a new message."';
    this.results.push(this.$refs.pEl.textContent);
    this.$nextTick(() => {
      this.results.push(this.$refs.pEl.textContent + ' (using $nextTick())');
    });
  }
}

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்க "Try it Yourself" பொத்தானைக் கிளிக் செய்யவும்.

மேலும் எடுத்துக்காட்டுகளை கீழே பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

$nextTick() மெத்தட் execute செய்வதற்கு முன் DOM update ஆக காத்திருக்கிறது.

நாம் this.$nextTick() ஐ தற்போதைய Vue கம்போனெண்டின் DOM update cycle முடிவடையும் வரை காத்திருக்க பயன்படுத்துகிறோம்.

Argument விளக்கம்
callback function Optional. வழங்கப்பட்ட callback function DOM update ஆன பிறகு இயக்கப்படும் (மேலே உள்ள எடுத்துக்காட்டைப் பார்க்கவும்). $nextTick() மெத்தட் argument இல்லாமலும் பயன்படுத்தப்படலாம் (கீழே உள்ள எடுத்துக்காட்டைப் பார்க்கவும்).

this.$nextTick() க்கு கூடுதலாக, ஒரு global nextTick() மெத்தடும் உள்ளது, இது ஒரு குறிப்பிட்ட கம்போனெண்டின் scope க்கு வெளியே இருந்துகூட DOM update ஆக காத்திருக்க பயன்படுத்தப்படலாம்.

💡 குறிப்பு:

Vue இல், ஒரு reactive variable மாற்றப்படும் போது, DOM உடனடியாக update செய்யப்படுவதில்லை. Vue இந்த மாற்றங்களை சேமிக்கிறது, மேலும் 'next tick' நடக்கும் போது அவற்றைப் பயன்படுத்துகிறது. இது performance ஐ மேம்படுத்துவதற்கும் Vue instance மற்றும் DOM க்கு இடையே consistency ஐ உறுதிப்படுத்துவதற்கும் ஆகும்.

மேலும் எடுத்துக்காட்டுகள்

எடுத்துக்காட்டு

முதல் எடுத்துக்காட்டில் உள்ள அதே முடிவை $nextTick() மெத்தடை asynchronous method இல் await prefix உடன் அழைப்பதன் மூலம் அடைய முடியும். இது 'next tick' நடக்கும் வரை குறியீட்டின் அடுத்த வரிகளை hold இல் வைக்கிறது.

<template>
  <h2>Example $nextTick() Method</h2>
  <p>Using "await $nextTick()", the next lines of code will also wait until the 'next tick' happens.
  </p>
  <div>
    <p ref="messageEl">{{ message }}</p>
    <button v-on:click.once="updateMsg">Update Message</button>
    <ol>
      <li v-for="x in results">{{ x }}</li>
    </ol>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: "Initial Message",
      results: []
    };
  },
  methods: {
    async updateMsg() {
      this.message = "Hello! This message is now updated.";
      this.results.push(this.$refs.messageEl.textContent);
      await this.$nextTick();
      this.results.push(this.$refs.messageEl.textContent + ' (after await $nextTick())');
    }
  }
};
</script>

<style scoped>
div {
  border: solid black 1px;
  padding: 10px;
}
</style>

Vue பயிற்சிகள்

இந்த டுடோரியலில் உள்ள பல அத்தியாயங்கள் உங்கள் அறிவு நிலையைச் சரிபார்க்கக்கூடிய பயிற்சியுடன் முடிகின்றன.

❓ $nextTick() மெத்தட் பற்றிய சரியான கூற்று எது?

$nextTick() DOM update ஆகும் வரை காத்திருக்கப் பயன்படுகிறது
✓ சரி! $nextTick() மெத்தட் Vue இன் reactive updates DOM இல் பிரதிபலிக்கும் வரை காத்திருக்கப் பயன்படுகிறது. இது DOM எலிமெண்டுகளை அணுகுவதற்கு முன் அவை update ஆகியுள்ளதை உறுதிப்படுத்துகிறது
$nextTick() உடனடியாக குறியீட்டை இயக்குகிறது
✗ தவறு! $nextTick() உடனடியாக குறியீட்டை இயக்காது. இது Vue இன் 'next tick' நடக்கும் வரை காத்திருக்கிறது, இது Vue reactive updates ஐ DOM இல் பிரதிபலிக்கும் பிறகு நடக்கும்
$nextTick() synchronous முறையில் மட்டுமே பயன்படுத்தப்படலாம்
✗ தவறு! $nextTick() synchronous மற்றும் asynchronous இரண்டு முறைகளிலும் பயன்படுத்தப்படலாம். callback function அல்லது await உடன் பயன்படுத்தலாம்
$nextTick() Vue இல் DOM update செய்கிறது
✗ தவறு! $nextTick() DOM update செய்வதில்லை. இது DOM ஏற்கனவே update செய்யப்பட்ட பிறகு குறியீட்டை இயக்க காத்திருக்கிறது. DOM update Vue இன் reactive system மூலம் நடக்கிறது
