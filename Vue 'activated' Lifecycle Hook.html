Vue 'activated' Lifecycle Hook

🏁 உதவிக்குறிப்பு:

உங்கள் Vue சேவையகத்தைப் பெறுங்கள்

activated ஹூக் அழைக்கப்படும் ஒவ்வொரு முறையும் பதிவு செய்ய activated வாழ்க்கைச் சுழற்சி ஹூக்கைப் பயன்படுத்துதல்.

Example

export default {
  mounted() {
    console.log("mounted");
    const liEl = document.createElement("li");
    liEl.innerHTML = "mounted";
    this.$refs.olEl.appendChild(liEl);
  },
  activated() {
    console.log("activated");
    const liEl = document.createElement("li");
    liEl.innerHTML = "activated";
    this.$refs.olEl.appendChild(liEl);
  }
}

வரையறை மற்றும் பயன்பாடு

activated வாழ்க்கைச் சுழற்சி ஹூக் ஒரு தற்காலிக சேமிப்பு கூறு DOM இல் செருகப்படும் போது இயங்கும்.

ஒரு கூறு உள்ளமைக்கப்பட்ட கூறைப் பயன்படுத்தி தற்காலிக சேமிப்பு செய்யப்படுகிறது.

ஒரு தற்காலிக சேமிப்பு கூறு உருவாக்கப்பட்ட பிறகு, அது DOM இலிருந்து பல முறை செருகப்படலாம் மற்றும் நீக்கப்படலாம், மேலும் ஒவ்வொரு முறையும் அத்தகைய தற்காலிக சேமிப்பு கூறு DOM இல் செருகப்படும் போது, activated வாழ்க்கைச் சுழற்சி ஹூக் அழைக்கப்படுகிறது.

⚠️ குறிப்பு:

activated மற்றும் mounted ஹுக்குகளுக்கு இடையே உள்ள வேறுபாடு என்னவென்றால், ஏற்கனவே இருக்கும் தற்காலிக சேமிப்பு கூறு DOM இல் செருகப்படும் போது mounted ஹூக் அழைக்கப்படாது.

❓ Vue கூறுகளில் activated வாழ்க்கைச் சுழற்சி ஹூக் எப்போது இயங்கும்?

கூறு முதன்முறையாக உருவாக்கப்படும் போது
✗ தவறு! இது created அல்லது mounted ஹூக்கிற்கு
கூறின் தரவு மாறும் போது
✗ தவறு! இது watch அல்லது computed பண்புகளுக்கு
கூறு அழிக்கப்படும் போது
✗ தவறு! இது unmounted ஹூக்கிற்கு
ஒரு தற்காலிக சேமிப்பு கூறு DOM இல் செருகப்படும் போது
✓ சரி! activated ஹூக் உடன் தற்காலிக சேமிப்பு கூறுகள் DOM இல் செருகப்படும் போது இயங்கும்
