Vue அறிமுகம்

Vue அறிமுகம்

Vue என்பது ஒரு JavaScript கட்டமைப்பாகும். இது ஒரு <script> குறிச்சொல்லுடன் ஒரு HTML பக்கத்தில் சேர்க்கப்படலாம்.

Vue, வழிமுறைகளுடன் HTML பண்புகளை விரிவுபடுத்துகிறது, மற்றும் வெளிப்பாடுகளுடன் HTML க்கு தரவை பிணைக்கிறது.

Vue ஒரு JavaScript கட்டமைப்பு

Vue என்பது JavaScript இல் எழுதப்பட்ட ஒரு முன்னணி JavaScript கட்டமைப்பாகும்.

Vue க்கு ஒத்த கட்டமைப்புகள் React மற்றும் Angular ஆகும், ஆனால் Vue மிகவும் இலகுவானது மற்றும் தொடங்குவது எளிதானது.

Vue ஒரு JavaScript கோப்பாக விநியோகிக்கப்படுகிறது, மற்றும் ஒரு script குறிச்சொல்லுடன் ஒரு வலைப்பக்கத்தில் சேர்க்கப்படலாம்:

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

ஏன் Vue கற்றுக்கொள்ள வேண்டும்?

  • இது எளிமையானது மற்றும் பயன்படுத்த எளிதானது.
  • இது எளிமையான மற்றும் சிக்கலான திட்டங்களையும் கையாள முடியும்.
  • அதன் வளர்ந்து வரும் புகழ் மற்றும் திறந்த-மூல சமூக ஆதரவு.
  • சாதாரண JavaScript இல் HTML மற்றும் JavaScript எவ்வாறு இணைக்கப்பட்டுள்ளது என்று எழுத வேண்டும், ஆனால் Vue இல் ஒரு இணைப்பு உள்ளது என்பதை உறுதி செய்து, மீதமுள்ளவற்றை Vue கவனிக்கும்.
  • இது ஒரு திறமையான வளர்ச்சி செயல்முறையை அனுமதிக்கிறது, இது டெம்ப்ளேட்-அடிப்படையிலான தொடரியல், இருதரப்பு தரவு பிணைப்பு மற்றும் ஒரு மையப்படுத்தப்பட்ட மாநில மேலாண்மை ஆகியவற்றைக் கொண்டுள்ளது.

💡 குறிப்பு:

இந்த புள்ளிகளில் சில புரிந்துகொள்வதற்கு கடினமாக இருந்தால், கவலைப்பட வேண்டாம், இந்த டுடோரியலின் முடிவில் நீங்கள் புரிந்துகொள்வீர்கள்.

விருப்பங்கள் API

Vue இல் குறியீட்டை எழுத இரண்டு வெவ்வேறு வழிகள் உள்ளன: விருப்பங்கள் API மற்றும் கலவை API.

அடிப்படை கருத்துகள் விருப்பங்கள் API மற்றும் கலவை API இரண்டிற்கும் ஒரே மாதிரியானவை, எனவே ஒன்றைக் கற்றுக்கொண்ட பிறகு, மற்றொன்றிற்கு எளிதாக மாறலாம்.

விருப்பங்கள் API இந்த டுடோரியலில் எழுதப்பட்டுள்ளது, ஏனெனில் இது தொடக்கநிலைக்கு மிகவும் உதவியாகவும், மிகவும் அறிமுகமான கட்டமைப்புடன் கருதப்படுகிறது.

விருப்பங்கள் API மற்றும் கலவை API இடையே உள்ள வேறுபாடுகளைப் பற்றி மேலும் அறிய இந்த டுடோரியலின் முடிவில் இந்த பக்கத்தைப் பாருங்கள்.

எனது முதல் பக்கம்

5 அடிப்படை படிகளில் நமது முதல் Vue வலைப்பக்கத்தை எவ்வாறு உருவாக்கலாம் என்பதை இப்போது கற்றுக்கொள்வோம்:

  1. ஒரு அடிப்படை HTML கோப்புடன் தொடங்கவும்.
  2. Vue இணைய ஒரு <div> குறிச்சொல்லை id="app" உடன் சேர்க்கவும்.
  3. Vue உடன் இணைப்பு ஒரு <script> குறிச்சொல்லை Vue க்கான இணைப்புடன் சேர்ப்பதன் மூலம் உலாவிக்கு எவ்வாறு கையாள வேண்டும் என்று கூறுங்கள்.
  4. Vue நிகழ்வு உள்ளே ஒரு <script> குறிச்சொல்லை சேர்க்கவும்.
  5. Vue நிகழ்வை <div id="app"> குறிச்சொல்லுடன் இணைக்கவும்.

இந்தப் படிகள் விரிவாக விவரிக்கப்பட்டுள்ளன, 'Try It Yourself' எடுத்துக்காட்டுடன் முழு குறியீடும் கீழே உள்ளது.

1 HTML பக்கம்

ஒரு எளிய HTML பக்கத்துடன் தொடங்கவும்:

<!DOCTYPE html>
<html lang="en">
<head>
  <title>My first Vue page</title>
</head>
<body>

</body>
</html>

2 ஒரு <div> சேர்க்கவும்

Vue க்கு உங்கள் பக்கத்தில் ஒரு HTML உறுப்பு இணைக்கப்பட வேண்டும்.

<body> குறிச்சொல்லுக்குள் ஒரு <div> குறிச்சொல்லை வைத்து, அதற்கு ஒரு id கொடுங்கள்:

<body>
  <div id="app"></div>
</body>

3 Vue க்கு இணைப்பு சேர்க்கவும்

எங்கள் Vue குறியீட்டை விளக்க உலாவியை உதவ, இந்த <script> குறிச்சொல்லைச் சேர்க்கவும்:

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

4 Vue நிகழ்வை சேர்க்கவும்

இப்போது எங்கள் Vue குறியீட்டை சேர்க்க வேண்டும்.

இது Vue நிகழ்வு என்று அழைக்கப்படுகிறது மற்றும் தரவு மற்றும் முறைகள் மற்றும் பிற விஷயங்களைக் கொண்டிருக்கலாம், ஆனால் இப்போது அது ஒரு செய்தியை மட்டுமே கொண்டுள்ளது.

இந்த <script> குறிச்சொல்லின் கடைசி வரியில் எங்கள் Vue நிகழ்வு <div id="app"> குறிச்சொல்லுடன் இணைக்கப்பட்டுள்ளது:

<div id="app"></div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<script>

  const app = Vue.createApp({
    data() {
      return {
        message: "Hello World!"
      }
    }
  })

 app.mount('#app')

</script>

5 உரை இடைச்செருகல் மூலம் 'message' காட்டு

இறுதியாக, நாம் உரை இடைச்செருகலைப் பயன்படுத்தலாம், இது தரவுக்கான ஒரு பொறிக்குறியாக இரட்டை சுருள் பிரேஸ்கள் {{ }} உடன் ஒரு Vue தொடரியல் ஆகும்.

<div id="app"> {{ message }} </div>

உலாவி {{ message }} ஐ Vue நிகழ்வுக்குள் உள்ள 'message' பண்புகளில் சேமிக்கப்பட்ட உரையுடன் மாற்றும்.

முழு எடுத்துக்காட்டு: எனது முதல் Vue பக்கம்!

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

<!DOCTYPE html>
<html lang="en">
<head>
  <title>My first Vue page</title>
</head>
<body>

  <div id="app">
    {{ message }}
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

  <script>
    const app = Vue.createApp({
      data() {
        return {
          message: "Hello World!"
        }
      }
    })

   app.mount('#app')

  </script>
</body>
</html>

இந்த குறியீட்டை 'Try it Yourself' பொத்தானுடன் சோதிக்கவும்.

உரை இடைச்செருகல்

உரை இடைச்செருகல் என்பது வலைப்பக்கத்தில் காட்ட Vue நிகழ்விலிருந்து உரை எடுக்கப்படும் போது ஆகும்.

உலாவி இந்தக் குறியீட்டை உள்ளே கொண்ட பக்கத்தைப் பெறுகிறது:

<div id="app"> {{ message }} </div>

பின்னர் உலாவி Vue நிகழ்வின் 'message' பண்புக்குள் உள்ள உரையைக் கண்டறிந்து Vue குறியீட்டை இதில் மொழிபெயர்க்கிறது:

<div id="app">Hello World!</div>

உரை இடைச்செருகலில் JavaScript

எளிமையான JavaScript வெளிப்பாடுகளையும் இரட்டை சுருள் பிரேஸ்கள் {{ }} உள்ளே எழுதலாம்.

எடுத்துக்காட்டு

div உறுப்புக்குள் செய்திக்கு ஒரு சீரற்ற எண்ணைச் சேர்க்க JavaScript தொடரியலைப் பயன்படுத்தவும்:

<div id="app">
  {{ message }} <br>
  {{'Random number: ' + Math.ceil(Math.random()*6) }}
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<script>

  const app = Vue.createApp({
    data() {
      return {
        message: "Hello World!"
      }
    }
  })

 app.mount('#app')

</script>

தொடங்குங்கள்

இந்த டுடோரியல் உங்களுக்கு Vue இன் அடிப்படைகளைக் கற்பிக்கும்.

இந்த டுடோரியலைப் பின்பற்ற HTML, CSS மற்றும் JavaScript ஆகியவற்றின் அடிப்படை முன்னறிவு அனுபவம் தேவை.

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களைச் சோதிக்கவும்

❓ பயிற்சி:

'message' தரவு பண்பு மதிப்பைக் காட்ட உரை இடைச்செருகலைப் பயன்படுத்தவும்.

<div id="app">
</div>
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        message: "Hello World!"
      }
    }
  })
  app.mount('#app')
</script>
<div id="app"> message </div>
✗ தவறு! இது வெறும் HTML உரை, Vue இடைச்செருகல் அல்ல
<div id="app"> {{ "Hello World!" }} </div>
✗ தவறு! இது நேரடி உரை, 'message' தரவு பண்பு அல்ல
<div id="app"> {{ message }} </div>
✓ சரி! இது Vue இடைச்செருகல் மூலம் 'message' தரவு பண்பைக் காட்டுகிறது
<div id="app"> ${message} </div>
✗ தவறு! இது Vue தொடரியல் அல்ல, இது JavaScript template literals
