Vue வாழ்க்கைச் சுழற்சி கொக்கிகள்

Vue பயன்பாட்டின் வாழ்க்கைச் சுழற்சியின் வெவ்வேறு நிலைகளில் குறியீட்டை இயக்க Vue இல் வாழ்க்கைச் சுழற்சி கொக்கிகள் பயன்படுத்தப்படுகின்றன.

வாழ்க்கைச் சுழற்சி கொக்கிகள் அட்டவணை

வாழ்க்கைச் சுழற்சி கொக்கி விளக்கம்
beforeCreate மற்ற எல்லா வாழ்க்கைச் சுழற்சி கொக்கிகளுக்கும் முன் நடக்கிறது
created கூறு துவக்கப்படுகிறது, மற்றும் கூறு நிகழ்வு பண்புகளை அணுகலாம்
beforeMount கூறு இன்னும் ஏற்றப்படவில்லை, எனவே DOM உறுப்புகளை அணுக முடியாது
mounted கூறு DOM மரத்தில் ஏற்றப்பட்டுள்ளது, எனவே DOM உறுப்புகளை அணுகலாம்
beforeUpdate Vue இன் எதிர்வினை அமைப்பு புதிய ரெண்டரிங் தேவைப்படும் மாற்றத்தைக் கண்டறியும் போது நடக்கிறது
updated DOM மரம் புதுப்பித்த பிறகு நடக்கிறது
beforeUnmount ஒரு கூறு DOM இலிருந்து நீக்கப்படுவதற்கு சற்று முன் நடக்கிறது
unmounted ஒரு கூறு DOM இலிருந்து நீக்கப்பட்ட பிறகு நடக்கிறது
errorCaptured ஒரு குழந்தை/வழித்தோன்றல் கூறில் பிழை ஏற்படும் போது நடக்கிறது
renderTracked ஒரு ரெண்டர் செயல்பாடு ஒரு எதிர்வினை கூறைக் கண்காணிக்க அல்லது கண்காணிக்க அமைக்கப்படும் போது நடக்கிறது
renderTriggered கண்காணிக்கப்பட்ட எதிர்வினை கூறில் மாற்றம் ஏற்படும் போது நடக்கிறது, இதனால் புதிய ரெண்டர் தூண்டப்படுகிறது
activated ஒரு கேச் செய்யப்பட்ட டைனமிக் கூறு சேர்க்கப்படும் போது நடக்கிறது (ஆனால் ஏற்கனவே DOM இல் உள்ளது)
deactivated ஒரு கேச் செய்யப்பட்ட டைனமிக் கூறு நீக்கப்படும் போது நடக்கிறது (ஆனால் DOM இலிருந்து அல்ல)
serverPrefetch சேவையக-பக்க ரெண்டரிங் (SSR) போது நடக்கிறது

💻 சான்றிதழ் வழங்கல்

எங்கள் சான்றிதழ் படிப்புகளுடன் உங்கள் திறமைகளைச் சரிபார்க்கவும்.

❓ பயிற்சி:

கீழே உள்ள வாழ்க்கைச் சுழற்சி கொக்கிகளில் எது கூறு DOM க்கு ஏற்றப்பட்ட பிறகு இயங்குகிறது மற்றும் DOM உறுப்புகளை அணுக அனுமதிக்கிறது?

beforeCreate
✗ தவறு! beforeCreate கூறு உருவாக்கத்திற்கு முன் இயங்குகிறது, DOM கிடைக்காது
created
✗ தவறு! created கூறு துவக்கப்பட்ட பிறகு இயங்குகிறது, ஆனால் DOM இன்னும் ஏற்றப்படவில்லை
mounted
✓ சரி! mounted கூறு DOM க்கு ஏற்றப்பட்ட பிறகு இயங்குகிறது, DOM உறுப்புகளை அணுக அனுமதிக்கிறது
beforeMount
✗ தவறு! beforeMount ஏற்றத்திற்கு முன் இயங்குகிறது, DOM உறுப்புகள் இன்னும் கிடைக்காது
