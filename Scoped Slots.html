Vue.js இல் கூறுகளிலிருந்து பெற்றோருக்கு தரவை அனுப்ப கற்றுக்கொள்ளுங்கள்

Scoped Slots

ஒரு Scoped slot கூறிலிருந்து உள்ளூர் தரவை வழங்குகிறது, இதனால் பெற்றோர் அதை எவ்வாறு render செய்வது எனத் தேர்வு செய்யலாம்.

Scoped Slot தரவுப் பாய்வு

தரவுப் பாய்வு வரைபடம்

கூறு தரவு

கூறில் உள்ளூர் தரவு

v-bind

தரவைப் பைண்ட் செய்கிறது

v-slot

தரவைப் பெறுகிறது

பெற்றோர்

தரவை render செய்கிறது

பெற்றோருக்கு தரவை அனுப்புதல்

உள்ளூர் தரவைப் பெற்றோருக்கு அனுப்ப கூறு slot இல் v-bind ஐப் பயன்படுத்துகிறோம்:

கூறு (SlotComp.vue)

<template>
  <slot v-bind:lclData="data"></slot>
</template>

<script>
  export default {
    data() {
      return {
        data: 'This is local data'
      }
    }
  }
</script>

பெற்றோர் (App.vue)

<slot-comp v-slot="dataFromSlot">
  <h2>{{ dataFromSlot.lclData }}</h2>
</slot-comp>

🔒 உள்ளூர் தரவு:

கூறின் உள்ளே இருக்கும் தரவு 'உள்ளூர்' என்று குறிப்பிடப்படலாம், ஏனெனில் இது v-bind உடன் பெற்றோருக்கு அனுப்பப்படாத வரை பெற்றோருக்கு அணுகக்கூடியதல்ல.

Scoped Slot இலிருந்து தரவைப் பெறுதல்

கூறில் உள்ள உள்ளூர் தரவு v-bind உடன் அனுப்பப்படுகிறது, மேலும் அது பெற்றோரில் v-slot உடன் பெறப்படலாம்:

எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

// App.vue:
<slot-comp v-slot="dataFromSlot">
  <h2>{{ dataFromSlot.lclData }}</h2>
</slot-comp>

🏷️ பெயரிடல்:

மேலே உள்ள எடுத்துக்காட்டில், 'dataFromSlot' என்பது Scoped slot இலிருந்து நாம் பெறும் தரவுப் பொருளைக் குறிக்க நாமே தேர்ந்தெடுக்கக்கூடிய ஒரு பெயர். 'lclData' பண்புகளைப் பயன்படுத்தி slot இலிருந்து உரை சரத்தைப் பெறுகிறோம், மேலும் இறுதியாக உரையை ஒரு <h2> டேக்கில் render செய்ய இடைச்செருகலைப் பயன்படுத்துகிறோம்.

வரிசையுடன் Scoped Slot

ஒரு Scoped slot v-for ஐப் பயன்படுத்தி ஒரு வரிசையிலிருந்து தரவை அனுப்பலாம், ஆனால் App.vue இல் உள்ள குறியீடு அடிப்படையில் ஒரே மாதிரியாக உள்ளது:

எடுத்துக்காட்டு

கூறு (SlotComp.vue)

<template>
  <slot
    v-for="x in foods"
    :key="x"
    :foodName="x"
  ></slot>
</template>

<script>
  export default {
    data() {
      return {
        foods: ['Apple','Pizza','Rice','Fish','Cake']
      }
    }
  }
</script>

பெற்றோர் (App.vue)

<slot-comp v-slot="food">
  <h2>{{ food.foodName }}</h2>
</slot-comp>

பொருள்களின் வரிசையுடன் Scoped Slot

ஒரு Scoped slot v-for ஐப் பயன்படுத்தி பொருள்களின் வரிசையிலிருந்து தரவை அனுப்பலாம்:

எடுத்துக்காட்டு

கூறு (SlotComp.vue)

<template>
  <slot
    v-for="x in foods"
    :key="x.name"
    :foodName="x.name"
    :foodDesc="x.desc"
    :foodUrl="x.url"
  ></slot>
</template>

<script>
  export default {
    data() {
      return {
        foods: [
          { name: 'Apple', desc: 'Apples are a type of fruit that grow on trees.', url: 'img_apple.svg' },
          { name: 'Pizza', desc: 'Pizza has a bread base with tomato sauce, cheese, and toppings on top.', url: 'img_pizza.svg' },
          { name: 'Rice', desc: 'Rice is a type of grain that people like to eat.', url: 'img_rice.svg' },
          { name: 'Fish', desc: 'Fish is an animal that lives in water.', url: 'img_fish.svg' },
          { name: 'Cake', desc: 'Cake is something sweet that tastes good but is not considered healthy.', url: 'img_cake.svg' }
       ]
      }
    }
  }
</script>

பெற்றோர் (App.vue)

<slot-comp v-slot="food">
  <hr>
  <h2>{{ food.foodName }}<img :src=food.foodUrl></h2>
  <p>{{ food.foodDesc }}</p>
</slot-comp>

Scoped Slot இலிருந்து நிலையான தரவு

ஒரு Scoped slot நிலையான தரவையும் அனுப்பலாம், அதாவது Vue நிகழ்வின் தரவுப் பண்புக்கு சொந்தமில்லாத தரவு.

நிலையான தரவை அனுப்பும் போது v-bind ஐப் பயன்படுத்துவதில்லை.

எடுத்துக்காட்டு

கூறு (SlotComp.vue)

<template>
  <slot
    staticText="This text is static"
    :dynamicText="text"
  ></slot>
</template>

<script>
  export default {
    data() {
      return {
        text: 'This text is from the data property'
      }
    }
  }
</script>

பெற்றோர் (App.vue)

<slot-comp v-slot="texts">
  <h2>{{ texts.staticText }}</h2>
  <p>{{ texts.dynamicText }}</p>
</slot-comp>

📝 நிலையான vs டைனமிக்:

கீழே உள்ள எடுத்துக்காட்டில், தரவு நிகழ்வுக்கு டைனமிக் முறையில் பைண்ட் செய்யப்படும் ஒரு உரையுடன் ஒரு நிலையான உரையை அனுப்புகிறோம், இதனால் வித்தியாசத்தைக் காணலாம்.

பெயரிடப்பட்ட Scoped Slots

Scoped slots பெயரிடப்படலாம்.

பெயரிடப்பட்ட Scoped slots ஐப் பயன்படுத்த, கூறுக்குள் உள்ள slots களை 'name' பண்புக்கூறுடன் பெயரிட வேண்டும்.

மேலும் பெயரிடப்பட்ட slot இலிருந்து தரவைப் பெற, v-slot உத்தியுடன் அல்லது சுருக்கெழுத்து # உடன் கூறைப் பயன்படுத்தும் பெற்றோரில் அந்தப் பெயரைக் குறிப்பிட வேண்டும்.

எடுத்துக்காட்டு

இந்த எடுத்துக்காட்டில், கூறு "leftSlot" ஐக் குறிப்பிடும் ஒரு முறையும், "rightSlot" ஐக் குறிப்பிடும் ஒரு முறையும் உருவாக்கப்படுகிறது.

கூறு (SlotComp.vue)

<template>
  <slot
    name="leftSlot"
    :text="leftText"
  ></slot>
  <slot
    name="rightSlot"
    :text="rightText"
  ></slot>
</template>

<script>
  export default {
    data() {
      return {
        leftText: 'This text belongs to the LEFT slot.',
        rightText: 'This text belongs to the RIGHT slot.'
      }
    }
  }
</script>

பெற்றோர் (App.vue)

<slot-comp #leftSlot="leftProps">
  <div>{{ leftProps.text }}</div>
</slot-comp>
<slot-comp #rightSlot="rightProps">
  <div>{{ rightProps.text }}</div>
</slot-comp>

மாற்று: பல டெம்ப்ளேட்டுகளுடன் ஒற்றைக் கூறு

மாற்றாக, கூறை ஒரு முறை மட்டும் உருவாக்கலாம், இரண்டு வெவ்வேறு "template" டேக்குகளுடன், ஒவ்வொரு "template" டேக்கும் வெவ்வேறு slot ஐக் குறிக்கிறது.

எடுத்துக்காட்டு

இந்த எடுத்துக்காட்டில், கூறு ஒரு முறை மட்டுமே உருவாக்கப்படுகிறது, ஆனால் இரண்டு "template" டேக்குகளுடன், ஒவ்வொன்றும் வெவ்வேறு slot ஐக் குறிக்கிறது.

SlotComp.vue முந்தைய எடுத்துக்காட்டில் உள்ளதைப் போலவே உள்ளது.

// App.vue:
<slot-comp>

  <template #leftSlot="leftProps">
    <div>{{ leftProps.text }}</div>
  </template>

  <template #rightSlot="rightProps">
    <div>{{ rightProps.text }}</div>
  </template>

</slot-comp>

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களை சோதிக்கவும்

❓ கூறின் slot இலிருந்து பெற்றோருக்கு தரவை வழங்க என்ன உத்திகள் தேவை?

கூறில் உள்ள உள்ளூர் தரவு ஒரு slot இலிருந்து ____ உடன் அனுப்பப்படுகிறது, மேலும் அது பெற்றோரில் ____ உடன் பெறப்படலாம்.

CompOne.vue

<slot 
:lclData="data"></slot>

App.vue

<comp-one 
:"dataFromSlot">
  <h2>{{ dataFromSlot.lclData }}</h2>
</comp-one>
v-model, v-receive
✗ தவறு! v-model இருதரப்பு தரவு பைண்டிங்கிற்கு, v-receive என்பது Vue இல் இல்லை
v-on, v-catch
✗ தவறு! v-on நிகழ்வுகளுக்கு, v-catch என்பது Vue இல் இல்லை
v-bind, v-slot
✓ சரி! v-bind தரவை slot இலிருந்து அனுப்புகிறது, v-slot பெற்றோரில் அதைப் பெறுகிறது
v-send, v-get
✗ தவறு! v-send மற்றும் v-get என்பவை Vue இல் இல்லை
