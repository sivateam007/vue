Vue 'data' Option

🎯 Example

ஒரு செய்தியை வைத்திருக்க data விருப்பத்திற்குள் ஒரு தரவு பண்பைப் பயன்படுத்துதல்.

export default {
  data() {
    return {
      msg: 'Hello World!'
    };
  }
};

Definition and Usage

data விருப்பம் என்பது உள்ளே உள்ள அனைத்து தரவு பண்புகளுடனும் ஒரு பொருளைத் திருப்பி அனுப்பும் ஒரு செயல்பாடாகும்.

data செயல்பாடு திருப்பி அனுப்பும் பொருளை this.$data உடன் அணுகலாம், மற்றும் ஒரு குறிப்பிட்ட தரவு பண்பு 'count' this.$data.count உடன் அணுகலாம், அல்லது வெறுமனே this.count.

$ அல்லது _ உடன் தொடங்கும் பெயரைக் கொண்ட தரவு பண்புகள் this.$data பொருள் மூலம் அணுகப்பட வேண்டும், அவை வேறு வழியில் அணுக முடியாது.

பயன்பாடு இயங்கத் தொடங்கிய பிறகு, this.$data ஐப் பயன்படுத்தி புதிய தரவு பண்புகளைச் சேர்க்க முடியும், ஆனால் இது பரிந்துரைக்கப்படுவதில்லை.
