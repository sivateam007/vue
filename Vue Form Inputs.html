இந்த டுடோரியலில் முன்பு Vue படிவங்கள் மற்றும் v-model பக்கங்களில் படிவ உள்ளீட்டு எடுத்துக்காட்டுகளைப் பார்த்துள்ளோம்.

இந்தப் பக்கம் Vue இல் உள்ள மேலும் படிவ உள்ளீட்டு எடுத்துக்காட்டுகளின் தொகுப்பாகும், ரேடியோ பொத்தான்கள், செக்பாக்ஸ்கள், டிராப்-டவுன் பட்டியல் மற்றும் சாதாரண உரை உள்ளீட்டு புலம் போன்றவை.

ரேடியோ பொத்தான்கள்

ஒரே தேர்வைச் சேர்ந்த ரேடியோ பொத்தான்கள் ஒரே பெயரைக் கொண்டிருக்க வேண்டும், இதனால் ஒரே ஒரு ரேடியோ பொத்தான் தேர்ந்தெடுக்கப்பட முடியும்.

Vue இல் உள்ள அனைத்து உள்ளீடுகளைப் போலவே, ரேடியோ பொத்தான் உள்ளீட்டு மதிப்பை v-model உடன் பிடிக்கிறோம், ஆனால் value பண்புக்கூறும் <input type="radio"> டேக்கில் வெளிப்படையாக அமைக்கப்பட வேண்டும்.

Vue படிவத்தில் ரேடியோ பொத்தான்களை எவ்வாறு பயன்படுத்தலாம்:

🎯 எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

App.vue:

<template>
  <h1>Radio Buttons in Vue</h1>
  <form @submit.prevent="registerAnswer">
    <p>What is your favorite animal?</p>
    <label>
      <input type="radio" name="favAnimal" v-model="inpVal" value="Cat"> Cat
    </label>
    <label>
      <input type="radio" name="favAnimal" v-model="inpVal" value="Dog"> Dog
    </label>
    <label>
      <input type="radio" name="favAnimal" v-model="inpVal" value="Turtle"> Turtle
    </label>
    <label>
      <input type="radio" name="favAnimal" v-model="inpVal" value="Moose"> Moose
    </label>
    <button type="submit">Submit</button>
  </form>
  <div>
    <h3>Submitted choice:</h3>
    <p id="pAnswer">{{ inpValSubmitted }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      inpVal: '',
      inpValSubmitted: 'Not submitted yet'
    }
  },
  methods: {
    registerAnswer() {
      if(this.inpVal) {
        this.inpValSubmitted = this.inpVal;
      }
    }
  }
}
</script>

டிராப்-டவுன் பட்டியல்

ஒரு டிராப்-டவுன் பட்டியல் <select> டேக் மற்றும் உள்ளே <option> டேக்குகளைக் கொண்டுள்ளது.

Vue இல் டிராப்-டவுன் பட்டியலைப் பயன்படுத்தும் போது, <select> டேக்கை v-model உடன் இணைக்க வேண்டும், மேலும் <option> டேக்குகளுக்கு மதிப்புகளை வழங்க வேண்டும்:

  <form @submit.prevent="registerAnswer">
    <label for="cars">Choose a car:</label>

  </form>

    <p id="pAnswer">{{ inpValSubmitted }}</p>
  </div>
</template>

<script>
export default {

    }
  },

      if(this.carSelected) {

      }

  }
}

Vue Form Inputs பயிற்சி

உள்ளீடு 'inpText' தரவு பண்புடன் இரு-வழி பைண்டிங் பெறும் வகையில் விடுபட்ட குறியீட்டை நிரப்பவும்.

✏️ Vue Form Inputs பயிற்சி

<template>
  <input type="text" _____>
  <p> {{ inpText }} </p>
</template>

<script>
  const app = Vue.createApp({
    data() {
      return {
        inpText: 'Initial text'
      }
    }
  })
  app.mount('#app')
</script>

கீழே உள்ள விருப்பங்களிலிருந்து சரியான பதிலைத் தேர்ந்தெடுக்கவும்:

value="inpText"
✗ தவறு! value பண்புக்கூறு நிலையான மதிப்பை அமைக்கிறது, ஆனால் இரு-வழி பைண்டிங் இல்லை
v-model="inpText"
✓ சரி! v-model Vue இல் இரு-வழி தரவு பைண்டிங்கிற்கான சரியான வழி. இது உள்ளீடு மற்றும் Vue தரவு பண்புக்கு இடையே இரு-வழி கம்யூனிகேஷனை உருவாக்குகிறது
:value="inpText"
✗ தவறு! :value ஒரு-வழி பைண்டிங்கை உருவாக்குகிறது (தரவிலிருந்து உள்ளீட்டிற்கு), ஆனால் இரு-வழி அல்ல
@input="inpText"
✗ தவறு! @input ஒரு நிகழ்வு கேட்பவர், ஆனால் இது தானாகவே இரு-வழி பைண்டிங்கை உருவாக்காது

படிக்க மட்டுமே படிவ உள்ளீடுகள்

படிவ உள்ளீடுகளில் v-model ஐப் பயன்படுத்துவது இரு-வழி பைண்டிங்கை உருவாக்குகிறது, அதாவது Vue தரவு நிகழ்வு மாறினால், உள்ளீட்டு value பண்புக்கூறும் மாறுகிறது.

படிக்க மட்டுமே படிவ உள்ளீடுகளுக்கு, <input type="file"> போன்றவை, value பண்புக்கூறை Vue தரவு நிகழ்விலிருந்து மாற்ற முடியாது, எனவே நாம் v-model ஐப் பயன்படுத்த முடியாது.

படிக்க மட்டுமே படிவ உள்ளீடுகளுக்கு, <input type="file"> போன்றவை, Vue தரவு நிகழ்வைப் புதுப்பிக்க ஒரு முறையை அழைக்க @change ஐப் பயன்படுத்த வேண்டும்:

<template>
  <h1>Input Type File</h1>
  <form @submit.prevent="registerAnswer">
    <label>Choose a file:
      <input @change="updateVal" type="file">
    </label>
    <button type="submit">Submit</button>
  </form>
  <div>
    <h3>Submitted answer:</h3>
    <p id="pAnswer">{{ inpValSubmitted }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      fileInp: null,
      inpValSubmitted: 'Not submitted yet'
    }
  },
  methods: {
    registerAnswer() {
      if(this.fileInp) {
        this.inpValSubmitted = this.fileInp;
      }
    },
    updateVal(e) {
      this.fileInp = e.target.value;
    }
  }
}
</script>

படிவ உள்ளீடு சிறந்த நடைமுறைகள்

v-model modifiers பயன்படுத்தவும்: .lazy, .number, .trim மோடிஃபையர்கள் உள்ளீடு செயலாக்கத்தை மேம்படுத்துகின்றன
சரியான உள்ளீட்டு வகைகளைப் பயன்படுத்தவும்: type="email", type="tel", type="number" உள்ளீட்டு சரிபார்ப்பு மற்றும் மொபைல் விசைப்பலகைகளை மேம்படுத்துகின்றன
தானியங்கி கவனம் மற்றும் சரிபார்த்தல்: autofocus மற்றும் required பண்புக்கூறுகளைப் பயன்படுத்தவும்
v-model ஐ படிக்க மட்டுமே உள்ளீடுகளில் பயன்படுத்த வேண்டாம்: file, image போன்ற உள்ளீடுகளுக்கு @change ஐப் பயன்படுத்தவும்
உள்ளீட்டு சரிபார்த்தல்: @blur, @change நிகழ்வுகளுடன் சரிபார்ப்பைச் செயல்படுத்தவும்
<!-- v-model modifiers examples -->
<input v-model.lazy="text">      <!-- Updates on change instead of input -->
<input v-model.number="age">     <!-- Converts to number -->
<input v-model.trim="username">  <!-- Trims whitespace -->

<!-- Form validation example -->
<input 
  type="email" 
  v-model="email"
  @blur="validateEmail"
  required
  :class="{ 'error': emailError }"
>
