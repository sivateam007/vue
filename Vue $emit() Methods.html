Vue $emit() Method

Vue $emit() மெத்தடைக் கற்றுக்கொள்ளுங்கள்

Vue $emit() மெத்தட்

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்
பொத்தான் கிளிக் செய்யப்படும் போது பேரென்ட் கம்போனென்டுக்கு ஒரு கஸ்டம் இவெண்டைத் தூண்ட $emit() மெத்தடைப் பயன்படுத்துதல்.
ChildComp.vue:
<template>
  <div>
    <h3>ChildComp.vue</h3>
    <p>Click the button to trigger the custom event up to the parent component using the $emit() method.</p>
    <button v-on:click="this.$emit('custom-event')">Trigger</button>
  </div>
</template>

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

பில்ட்-இன் $emit() மெத்தட் ஒரு கஸ்டம் இவெண்டைத் தூண்டுகிறது, இது பேரென்ட் கம்போனென்டுக்கு தொடர்புகொள்வதற்குப் பயன்படுகிறது.

வாதம் விளக்கம்
கஸ்டம் இவெண்ட் பெயர் டிஃபால்ட். இந்த முதல் வாதம் $emit() மெத்தட் மூலம் தூண்டப்படும் கஸ்டம் இவெண்டின் பெயர்.
மேலும் வாதங்கள் ஆப்ஷனல். கஸ்டம் இவெண்டுடன் ஒன்று அல்லது அதற்கு மேற்பட்ட வாதங்கள் ஒரு பேலோடாக அனுப்பப்படலாம். (கீழே உள்ள எடுத்துக்காட்டு 1 மற்றும் 2 ஐப் பார்க்கவும்.)

emits ஆப்ஷன் கம்போனென்ட் எதை emit செய்கிறது என்பதை ஆவணப்படுத்த பயன்படுத்தப்படலாம். emits ஆப்ஷனைப் பயன்படுத்துவது வாசிப்புத்திறனை மேம்படுத்துகிறது, ஆனால் இது தேவையில்லை. (கீழே உள்ள எடுத்துக்காட்டு 3 ஐப் பார்க்கவும்.)

Props எதிர் திசையில் தொடர்புகொள்வதற்குப் பயன்படுகிறது: பேரென்ட் கம்போனென்டிலிருந்து குழந்தை கம்போனென்டுக்கு கீழே. டுடோரியலில் props பற்றி மேலும் படிக்கவும்.

மேலும் எடுத்துக்காட்டுகள்

எடுத்துக்காட்டு 1

'message-sent' கஸ்டம் இவெண்டுடன், பேரென்ட் கம்போனென்டுக்கு ஒரு செய்தியை அனுப்ப $emit() மெத்தடைப் பயன்படுத்துதல்.

ChildComp.vue:
<template>
  <div>
    <h3>ChildComp.vue</h3>
    <p>Write something, and send the message up to the parent component using the $emit() method.</p>
    <input type="text" v-model="message" placeholder="write something..">
    <button v-on:click="send">Send</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: null
    }
  },
  methods: {
    send() {
      this.$emit('message-sent',this.message);
    }
  }
}
</script>

  padding: 10px;

  margin-top: 20px;
}
input {
  display: block;
  margin-bottom: 15px;
}
</style>

எடுத்துக்காட்டு 2

பேரென்ட் கம்போனென்டுக்கு ஒரு தயாரிப்பு பெயர் மற்றும் ரேட்டிங்கை அனுப்ப $emit() மெத்தடைப் பயன்படுத்துதல்.

ChildComp.vue:
<template>
  <div>
    <h3>ChildComp.vue</h3>
    <p>Rate a product:</p>
    <input type="text" v-model="productName" placeholder="Product name.." ref="inpName">
    <input type="number" v-model="productRating" placeholder="Rating 1 to 10..">
    <button v-on:click="send">Register</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      productName: null,
      productRating: null
    }
  },
  methods: {
    send() {
      this.$emit('message-sent',this.productName,this.productRating);
      this.productName = null;
      this.productRating = null;
      this.$refs.inpName.focus();
    }
  },
  mounted() {
    this.$refs.inpName.focus();
  }
}
</script>

<style scoped>
div {
  border: solid black 1px;
  padding: 10px;
  max-width: 350px;
  margin-top: 20px;
}
input {
  display: block;
  margin-bottom: 15px;
}
</style>

எடுத்துக்காட்டு 3

கம்போனென்ட் $emit() மெத்தட் மூலம் எதை emit செய்கிறது என்பதை ஆவணப்படுத்த emits ஆப்ஷனைப் பயன்படுத்துதல். இது தேவையில்லை, ஆனால் இது வாசிப்புத்திறனை மேம்படுத்துகிறது.

ChildComp.vue:
<template>
  <div>
    <h3>ChildComp.vue</h3>
    <p>Click the button to trigger the custom event up to the parent component using the $emit() method.</p>
    <button v-on:click="this.$emit('custom-event')">Trigger</button>
  </div>
</template>

<script>
export default {
  emits: ['custom-event']
}
</script>

<style scoped>
div {
  border: solid black 1px;
  padding: 10px;
  max-width: 350px;
  margin-top: 20px;
}
</style>

❓ பயிற்சி:

பேரென்ட் கம்போனென்டுக்கு கஸ்டம் இவெண்டைத் தூண்ட பயன்படுத்தப்படும் Vue இன் பில்ட்-இன் மெத்தட் எது?

$on()
✗ தவறு! $on() மெத்தட் இவெண்டுகளைக் கேட்கப் பயன்படுகிறது, தூண்ட அல்ல

✗ தவறு! $props() மெத்தட் props ஐ அணுகப் பயன்படுகிறது, இவெண்டுகளைத் தூண்ட அல்ல
$emit()
✓ சரி! $emit() மெத்தட் பேரென்ட் கம்போனென்டுக்கு கஸ்டம் இவெண்டுகளைத் தூண்டப் பயன்படுகிறது
$trigger()
✗ தவறு! Vue இல் $trigger() என்ற பில்ட்-இன் மெத்தட் இல்லை
