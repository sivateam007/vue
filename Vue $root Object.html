Vue $root Object

Vue $root பொருளைப் பற்றி அறிந்து கொள்ளுங்கள்

Vue $root பொருள்

$root பொருள் முழு Vue பயன்பாட்டின் ரூட் கூறின் Vue உதாரணத்தைக் குறிக்கிறது.

$root பொருள் ரூட் கூறில் பயன்படுத்தப்பட்டால், அது அந்த கூற்றின் உதாரணத்தையே குறிக்கும்.

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

Vue பயன்பாட்டின் ரூட் கூறில் உள்ள 'text' தரவுப் பண்பை மாற்ற குழந்தை கூறில் $root பொருளைப் பயன்படுத்துவது.

Example

<template>
  <div>
    <h3>Change Text</h3>
    <p>Click the button to toggle the text in the PRE tag of the root component.</p>
    <button v-on:click="this.$root.text='Hello!'">Change text in root</button>
  </div>
</template>

💡 குறிப்பு:

கீழே மேலும் எடுத்துக்காட்டுகளைப் பார்க்கவும்.

வரையறை மற்றும் பயன்பாடு

$root பொருள் முழு Vue பயன்பாட்டின் ரூட் கூறின் Vue உதாரணத்தைக் குறிக்கிறது.

$root பொருள் ரூட் கூறில் பயன்படுத்தப்பட்டால், அது அந்த கூற்றின் உதாரணத்தையே குறிக்கும்.

கூறு மர அமைப்பில் மிகவும் கீழே இருக்கும் குழந்தை கூறிலிருந்தும் ரூட் உதாரணத்தை நேரடியாக அணுக $root பொருளைப் பயன்படுத்தலாம், முறைகளை அழைக்க, தரவுப் பண்புகளைப் படிக்க அல்லது மாற்றியமைக்க, மற்றும் பல.

⚠️ கவனிக்க:

Vue கூறுகளுக்கு இடையேயான தொடர்புக்கு props/emit அல்லது provide/inject ஐப் பயன்படுத்துவதைக் கவனியுங்கள், ஏனெனில் இவை வெளிப்படையாக வரையறுக்கப்பட்ட தொடர்பு வழிகளைக் கொண்ட குறியீடு பராமரிக்க எளிதானது.

மேலும் எடுத்துக்காட்டுகள்

Example

கூறு மர அமைப்பில் ஒன்றுக்கு மேற்பட்ட நிலை மேலே உள்ள ரூட் கூறில் <p> குறியின் வண்ணத்தை மாற்ற குழந்தை கூறில் $root பொருளைப் பயன்படுத்துவது.

Template

<template>
  <div>
    <h4>Grand Child Component</h4>
    <p>Click the button to toggle the color of the P tag in the root component.</p>
    <button v-on:click="this.$root.color='lightgreen'">Change color in root</button>
  </div>
</template>
