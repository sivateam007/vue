Vue Directives

HTML குறிக்கு கூடுதல் செயல்பாட்டை வழங்கும் v- முன்னொட்டுடன் கூடிய சிறப்பு HTML பண்புகள் Vue directives

Vue Directives

Vue directives என்பது v- முன்னொட்டுடன் கூடிய சிறப்பு HTML பண்புகள் ஆகும், அவை HTML குறிக்கு கூடுதல் செயல்பாட்டை வழங்குகின்றன.

Vue directives மாறும் மற்றும் எதிர்வினை பயனர் இடைமுகங்களை உருவாக்க Vue உதாரணத்துடன் இணைகின்றன.

Vue உடன், பாரம்பரிய JavaScript முறைகளுடன் ஒப்பிடும்போது பதிலளிக்கும் பக்கங்களை உருவாக்குவது மிகவும் எளிதானது மற்றும் குறைவான குறியீடு தேவைப்படுகிறது.

Different Vue Directives

இந்த டுடோரியலில் நாம் பயன்படுத்தும் வெவ்வேறு Vue directives கீழே பட்டியலிடப்பட்டுள்ளன.

Directive விவரங்கள்
v-bind Vue உதாரணத்திற்குள் ஒரு தரவு மாறிக்கு HTML குறியில் ஒரு பண்பை இணைக்கிறது.
v-if ஒரு நிபந்தனையைப் பொறுத்து HTML குறிகளை உருவாக்குகிறது. v-else-if மற்றும் v-else directives v-if directive உடன் சேர்ந்து பயன்படுத்தப்படுகின்றன.
v-show ஒரு நிபந்தனையைப் பொறுத்து ஒரு HTML உறுப்பு காணப்பட வேண்டுமா இல்லையா என்பதைக் குறிப்பிடுகிறது.
v-for Vue உதாரணத்தில் ஒரு வரிசையின் அடிப்படையில் for-loop ஐப் பயன்படுத்தி குறிகளின் பட்டியலை உருவாக்குகிறது.
v-on ஒரு HTML குறியில் ஒரு நிகழ்வை JavaScript வெளிப்பாடு அல்லது Vue உதாரண முறையுடன் இணைக்கிறது. நிகழ்வு-மாற்றிகள் பயன்படுத்துவதன் மூலம் ஒரு குறிப்பிட்ட நிகழ்வுக்கு எங்கள் பக்கம் எவ்வாறு பதிலளிக்க வேண்டும் என்பதையும் நாம் வரையறுக்கலாம்.
v-model <form>, <input> மற்றும் <button> போன்ற குறிகளுடன் HTML படிவங்களில் பயன்படுத்தப்படுகிறது. ஒரு உள்ளீட்டு உறுப்புக்கும் Vue உதாரண தரவு பண்புக்கும் இடையே இருதரப்பு பிணைப்பை உருவாக்குகிறது.

Example: The v-bind Directive

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

<div> உறுப்பை எந்த வகுப்புடன் இணைக்க வேண்டும் என்பதை Vue உதாரணத்திலிருந்து உலாவி கண்டுபிடிக்கிறது.

<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    .pinkBG {
      background-color: lightpink;
    }
  </style>
</head>
<body>

  <div id="app">
    <div v-bind:class="vueClass"></div>
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <script>
    const app = Vue.createApp({
      data() {
        return {
          vueClass: "pinkBG"
        }
      }
    })
    app.mount('#app')
  </script>
</body>
</html>

💡 குறிப்பு:

மேலே உள்ள எடுத்துக்காட்டு Vue குறியீடு இல்லாமல் மிகவும் எளிமையாக எழுதப்படலாம், ஆனால் பொறுமையாக இருங்கள். பதிலளிக்கும் பக்கங்களை உருவாக்கும் போது Vue இன் உண்மையான நன்மைகள் பிந்தைய எடுத்துக்காட்டுகளில் காணப்படும்.
