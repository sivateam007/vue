Vue 'methods' விருப்பத்தைக் கற்றுக்கொள்ளுங்கள்

Vue 'methods' Option

🎯 Example

ஒரு செய்தியை மாற்ற methods விருப்பத்திற்குள் ஒரு முறையைப் பயன்படுத்துதல்.

export default {
  data() {
    return {
      msg: 'Hello World!',
      showMsg: false
    };
  },
  methods: {
    toggleMsg() {
      this.showMsg = !this.showMsg;
    }
  }
};

Definition and Usage

methods விருப்பம் என்பது Vue உதாரணத்தில் அறிவிக்கப்பட்ட அனைத்து முறைகளையும் கொண்ட ஒரு பொருளாகும்.

முறைகளை நேரடியாக (this முக்கிய சொல்லில்லாமல்) ஒரு Vue பயன்பாட்டின் <template> இலிருந்து அழைக்க முடியும், எடுத்துக்காட்டாக ஒரு நிகழ்வு நடக்கும்போது ஒரு முறை இயக்கப்படுவதை அமைக்கும் போது, v-on வழிகாட்டியைப் பயன்படுத்தி.

Vue உதாரணத்திற்குள் இருந்து ஒரு முறையை அழைக்க this முக்கிய சொல் பயன்படுத்தப்பட வேண்டும், எடுத்துக்காட்டாக ஒரு முறை மற்றொரு முறையால் அழைக்கப்படும் போது.

⚠️ குறிப்பு:

முறைகளை அறிவிக்கும் போது அம்புக்குறி செயல்பாடுகள் தவிர்க்கப்பட வேண்டும், ஏனெனில் this முக்கிய சொல்லைப் பயன்படுத்தி அத்தகைய செயல்பாட்டிற்குள் இருந்து Vue உதாரணத்தை அணுக முடியாது.

Vue Exercises

Vue 'methods' விருப்பத்தைப் பற்றிய உங்கள் அறிவைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue.js இல், methods விருப்பத்திற்குள் முறைகளை வரையறுக்க சிறந்த வழி எது?

சாதாரண செயல்பாடுகளைப் பயன்படுத்தவும், this சரியான Vue உதாரணத்தைக் குறிக்கும்
✓ சரி! சாதாரண செயல்பாடுகள் Vue முறைகளுக்கு சரியானவை. அவை Vue உதாரணத்தை this மூலம் அணுக அனுமதிக்கின்றன மற்றும் தரவு பண்புகள் மற்றும் பிற முறைகளுடன் சரியாக வேலை செய்கின்றன
அம்புக்குறி செயல்பாடுகளைப் பயன்படுத்தவும், அவை குறுகிய தொடரியல் கொண்டவை
✗ தவறு! அம்புக்குறி செயல்பாடுகள் Vue முறைகளுக்குப் பொருந்தாது. அவை தங்கள் சொந்த this மதிப்பைப் பிணைக்கின்றன, இது Vue உதாரணத்தைக் குறிக்காது
வெளிப்படையாகப் பிணைக்கப்பட்ட செயல்பாடுகளைப் பயன்படுத்தவும், .bind(this) உடன்
✗ தவறு! சாதாரண செயல்பாடுகள் போதுமானவை. Vue முறைகளுக்கு வெளிப்படையான பைண்டிங் தேவையில்லை, ஏனெனில் Vue இது தானாகவே கையாளுகிறது
ஒருங்கிணைந்த செயல்பாடுகளைப் பயன்படுத்தவும், அவை ஒரு வரிசையில் வரையறுக்கப்படுகின்றன
✗ தவறு! முறைகள் ஒரு வரிசையில் வரையறுக்கப்படுவதில்லை. அவை ஒரு பொருளில் வரையறுக்கப்படுகின்றன, ஒவ்வொரு முறையும் ஒரு செயல்பாடாக இருக்கும்
