Vue 'key' Attribute

Vue இல் 'key' பண்பைக் கற்றுக்கொள்ளுங்கள்

Vue 'key' பண்பு

Vue.js இல் 'key' பண்பு v-for டைரக்டிவுடன் பயன்படுத்தப்படும் மிக முக்கியமான பண்புகளில் ஒன்றாகும். இது Vue க்கு v-for உடன் உருவாக்கப்பட்ட கூறுகளை சரியாக வேறுபடுத்த உதவுகிறது.

🔑 முக்கிய புள்ளி:

key பண்பு இல்லாமல், வரிசை மாற்றப்பட்டால் கூறு பண்புகள் கலக்கப்படலாம். key பண்பு Vue க்கு ஒவ்வொரு கூறையும் தனித்துவமாக அடையாளம் காண உதவுகிறது.

எடுத்துக்காட்டு

உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

key பண்பு food-item கூறுகளில் v-for உடன் உருவாக்கப்பட்ட ஒவ்வொரு எலிமென்ட்டையும் தனித்துவமாக அடையாளம் காண பயன்படுகிறது.

<food-item
  v-for="x in foods"
  :key="x.name"
  :food-name="x.name"
  :food-desc="x.desc"
  :is-favorite="x.favorite"
/>

இந்த எடுத்துக்காட்டில், :key="x.name" ஒவ்வொரு food-item கூறுக்கும் தனித்துவமான அடையாளத்தை வழங்குகிறது. இது Vue க்கு கூறுகளை கண்காணிக்கவும், திறம்பட புதுப்பிக்கவும் உதவுகிறது.

Vue 'key' பண்பு பயிற்சி

Vue இல் 'key' பண்பு பற்றிய உங்கள் புரிதலைச் சோதிக்க இந்த பயிற்சியை முயற்சிக்கவும்.

❓ Vue இல் v-for லூப்புடன் 'key' பண்பு பயன்படுத்தப்படாத போது என்ன நடக்கும்?

வேகமான ரெண்டரிங் நடக்கும்
✗ தவறு! key இல்லாமல், Vue கூறுகளை திறம்பட புதுப்பிக்க முடியாது, இது வேகத்தை குறைக்கும்
கூறு பண்புகள் கலக்கப்படலாம்
✓ சரி! key பண்பு இல்லாமல், வரிசை மாற்றப்பட்டால் கூறு பண்புகள் கலக்கப்படலாம்
கூறுகள் தானாகவே அழிக்கப்படும்
✗ தவறு! கூறுகள் அழிக்கப்படாது, ஆனால் தவறாக புதுப்பிக்கப்படலாம்
Vue செயல்படுத்துவதை நிறுத்தும்
✗ தவறு! Vue இன்னும் செயல்படும், ஆனால் எதிர்பாராத நடத்தையுடன்
