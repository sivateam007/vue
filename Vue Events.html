Vue நிகழ்வுகள்

Vue இல் நிகழ்வு கையாளுதல் v-on வழிமுறையுடன் செய்யப்படுகிறது, இதனால் ஒரு பொத்தான் கிளிக் செய்யப்படும் போது எதையாவது நடக்கச் செய்யலாம்.

நிகழ்வு கையாளுதல் என்பது ஒரு குறிப்பிட்ட நிகழ்வு நடக்கும்போது ஒரு குறிப்பிட்ட குறியீட்டை இயக்க HTML உறுப்புகள் அமைக்கப்படும் போது ஆகும்.

Vue இல் நிகழ்வுகள் பயன்படுத்த எளிதானவை மற்றும் எங்கள் பக்கத்தை உண்மையில் எதிர்வினையாற்றும் வகையில் ஆக்கும்.

Vue முறைகள் ஒரு நிகழ்வு நடக்கும்போது இயக்க அமைக்கப்படும் குறியீடு ஆகும்.

v-on திருத்திகளுடன், ஒரு நிகழ்விற்கு எவ்வாறு எதிர்வினை செய்வது என்பதை மேலும் விரிவாக விவரிக்கலாம்.

நிகழ்வுகளுடன் தொடங்குங்கள்

ஒரு காட்டில் மூஸ்களை எண்ண ஒரு பொத்தானைக் கிளிக் செய்வது எப்படி என்பதைக் காட்ட ஒரு எடுத்துக்காட்டுடன் தொடங்குவோம்.

எங்களுக்கு தேவை:

  • ஒரு பொத்தான்
  • 'click' நிகழ்வைக் கேட்க <button> குறிச்சொல்லில் v-on
  • மூஸ்களின் எண்ணிக்கையை அதிகரிக்கும் குறியீடு
  • மூஸ்களின் எண்ணிக்கையை வைத்திருக்க Vue நிகழ்வில் ஒரு பண்பு (மாறி)
  • அதிகரித்த மூஸ்களின் எண்ணிக்கையைக் காட்ட இரட்டை சுருள் பிரேஸ்கள் {{}}
எடுத்துக்காட்டு

காட்டில் ஒரு மூஸை எண்ண பொத்தானைக் கிளிக் செய்யவும். பொத்தான் கிளிக் செய்யப்படும் ஒவ்வொரு முறையும் count பண்பு அதிகரிக்கிறது.

<div id="app">
  <img src="img_moose.jpg">
  <p>{{ "Moose count: " + count }}</p>
  <button v-on:click="count++">Count moose</button>
</div>

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  const app = Vue.createApp({
    data() {
      return {
        count: 0
      }
    }
  })
 app.mount('#app')
</script>

💻 உங்கள் சொந்த Vue சேவையகத்தைப் பெறுங்கள்

📝 குறிப்பு:

Vue உடன் வரும் ஒரு நன்மை என்னவென்றால், <p> குறிச்சொல்லில் உள்ள மூஸ்களின் எண்ணிக்கை தானாகவே புதுப்பிக்கப்படுகிறது. வெற்று JavaScript உடன், கூடுதல் குறியீட்டு வரியுடன் பயனர் பார்க்கும் எண்ணைப் புதுப்பிக்க வேண்டும்.

நிகழ்வுகள்

குறியீட்டை இயக்குவதற்கான தூண்டுதல்களாகப் பயன்படுத்தக்கூடிய பல நிகழ்வுகள் உள்ளன, மிகவும் பொதுவானவற்றில் சில: 'click', 'mouseover', 'mouseout', 'keydown' மற்றும் 'input'.

click

கிளிக் செய்யும் போது

mouseover

சுட்டி மேலே செல்லும் போது

mouseout

சுட்டி வெளியே செல்லும் போது

keydown

விசை அழுத்தும் போது

input

உள்ளீடு மாறும் போது

பயன்படுத்த நிகழ்வுகளின் முழுமையான பட்டியலுக்கு எங்கள் HTML DOM நிகழ்வுகள் பக்கத்தைப் பார்க்கலாம்.

'v-on'

v-on வழிமுறை பயனர் என்ன செய்கிறார் என்பதற்கு எதிர்வினையாற்றும் பக்கங்களை உருவாக்க அனுமதிக்கிறது.

Vue v-on எந்த நிகழ்வைக் கேட்க வேண்டும், மற்றும் அந்த நிகழ்வு நடக்கும்போது என்ன செய்ய வேண்டும் என்பதை உலாவிக்குச் சொல்வதன் மூலம் வேலை செய்கிறது.

முறைகள்

ஒரு நிகழ்வு நடக்கும்போது மிகவும் சிக்கலான குறியீட்டை இயக்க விரும்பினால், குறியீட்டை ஒரு Vue முறையில் வைத்து, இந்த முறையை HTML பண்பிலிருந்து குறிப்பிடலாம், இது போன்றது:

<p v-on:click="changeColor">Click me</p>

நிகழ்வு திருத்திகள்

v-on மற்றும் Vue முறைகளுக்கு கூடுதலாக, நிகழ்வு திருத்திகள் என்று அழைக்கப்படுவதைப் பயன்படுத்தலாம், இது ஒரு நிகழ்வை மாற்றியமைக்கிறது, எடுத்துக்காட்டாக ஒரு பக்கம் ஏற்றப்பட்ட பிறகு ஒரு முறை மட்டுமே நடக்கும், அல்லது 'submit' போன்ற ஒரு நிகழ்வை மாற்றியமைக்க ஒரு படிவத்தை சமர்ப்பிப்பதைத் தடுக்க.

மேலும் அறிக

நாம் பார்க்க முடிந்தால், Vue இல் நிகழ்வுகளைப் பயன்படுத்த கற்றுக்கொள்ள வேண்டிய மூன்று நுட்பங்கள் உள்ளன:

  1. Vue v-on வழிமுறை
  2. Vue முறைகள்
  3. Vue v-on திருத்திகள்

🚀 உதவிக்குறிப்பு:

இந்த டுடோரியலைத் தொடர்ந்து நிகழ்வு கையாளுதலுக்கான இந்த நுட்பங்களைப் பற்றி மேலும் அறிய 'அடுத்தது' பொத்தானைக் கிளிக் செய்யவும்.

Vue பயிற்சிகள்

பயிற்சிகளுடன் உங்களைச் சோதிக்கவும்

❓ பயிற்சி:

விடுபட்ட பகுதியை நிரப்பவும்.

Vue இல், நிகழ்வுகள் ______ வழிமுறையுடன் கையாளப்படுகின்றன.

v-if
✗ தவறு! v-if நிபந்தனை ரெண்டரிங்கிற்குப் பயன்படுகிறது, நிகழ்வு கையாளுதலுக்கு அல்ல
v-bind
✗ தவறு! v-bind பண்பு பிணைப்புக்குப் பயன்படுகிறது, நிகழ்வு கையாளுதலுக்கு அல்ல
v-on
✓ சரி! v-on நிகழ்வுகளைக் கையாளப் பயன்படுகிறது, எ.கா. v-on:click
v-for
✗ தவறு! v-for வளையங்களுக்குப் பயன்படுகிறது, நிகழ்வு கையாளுதலுக்கு அல்ல
